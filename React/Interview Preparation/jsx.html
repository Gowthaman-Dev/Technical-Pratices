JSX – 360° Preparation

1️⃣ Basic Definition Angle

❓ What is JSX?

JSX stands for JavaScript XML.

It is a syntax extension that allows us to write HTML-like code inside JavaScript.

Key Points:

  • Used in React
  • Looks like HTML
  • Transpiled to React.createElement()
  • Makes UI code readable

2️⃣ Why JSX Angle

❓ Why do we use JSX?

  • Makes UI code easier to read
  • Combines HTML + JS in one place
  • Improves developer productivity
  • Better error handling

3️⃣ Technical Deep Angle

❓ Is JSX valid JavaScript?

No. Browsers do not understand JSX directly.

It is converted into JavaScript using Babel.

Example:

JSX:

<h1>Hello</h1>

Converted to:

React.createElement("h1", null, "Hello")

🔥 Interviewers love this answer.

4️⃣ JSX Syntax Rules Angle

❓ What are JSX syntax rules?

Important Rules:

  • Must return single parent element
  • Use className instead of class
  • Use htmlFor instead of for
  • Tags must be closed
  • Use camelCase for attributes

Example:

<div className="box">
   <input type="text" />
</div>

5️⃣ Expressions Inside JSX Angle

❓ Can we use JavaScript inside JSX?

Yes. We can use JavaScript expressions inside curly braces {}.

Example:

const name = "Rocky";
<h1>Hello {name}</h1>

6️⃣ What Can Be Written Inside {} ?

Allowed:

  • Variables
  • Math operations
  • Function calls
  • Ternary operators

Example:

<p>{5 + 5}</p>
<p>{isLoggedIn ? "Welcome" : "Login"}</p>

❌ Not Allowed:

  • if statements
  • for loops

Instead use:

  • Ternary operator
  • map() function

7️⃣ Conditional Rendering Angle

❓ How do you write condition in JSX?

Use ternary:

{isAdmin ? <AdminPanel /> : <UserPanel />}

Or logical AND:

{isLoggedIn && <Dashboard />}

8️⃣ Looping in JSX Angle

❓ How to render list in JSX?

Use map():

{users.map(user => (
   <p key={user.id}>{user.name}</p>
))}

Important:

Always use unique key prop.

9️⃣ Difference Between HTML and JSX

HTML JSX
class className
for htmlFor

🔟 Fragment Angle

❓ What if I don’t want extra div?

Use Fragment:

<>
   <h1>Hello</h1>
   <p>World</p>
</>

1️⃣1️⃣ Real Interview Trap Question

❓ Why can't we use if statement directly in JSX?

Because JSX expects expressions, not statements.

if is a statement, but ternary is an expression.
