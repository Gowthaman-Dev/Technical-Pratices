2.What is the difference between React and Angular?


->React

React is a JavaScript library for building UI using components. It uses virtual DOM, supports one-way data binding, offers high flexibility, and we can choose our own packages like routing, state management, etc.


Tamil Explanation

React is JS UI library. Idhu very flexible, component based, virtual DOM use pannum, so speed & performance nalla irukum. Neenga ungala venumnaana library-ah choose pannalam.


->Angular


Angular is a complete framework built using TypeScript. It has two-way data binding, in-built features (routing, forms, http), and follows strong structure but has a steeper learning curve.


Tamil Explanation


Angular one full framework, ellame in-built ah kudukkum (routing, http, forms). Two-way data binding irukum and TypeScript mandatory. Learning konjam hard ana full structure irukum.




3: What is TypeScript?


TypeScript is a superset of JavaScript that adds static typing and compiles into normal JavaScript. It helps catch errors during development and makes code more predictable and maintainable.

Node.js is used in web development because it allows JavaScript to run on the server-side, not just in the browser. It enables fast, scalable, real-time applications, handles multiple requests efficiently using non-blocking I/O, and provides access to npm, which offers thousands of useful packages for development.
