Countdown Timer

Create a countdown timer
that updates every second

// Question Meaning என்ன?

10 இல் start ஆகி → 9, 8, 7... என்று கணக்கிட்டு → 0 ஆனதும் "Time's Up" காட்டு. ஒவ்வொரு second-லயும் number மாறணும்.

STEP 01

HTML Body ஏன்?

<button onclick="gowtham()">Click Me</button>
<p id="timer">10</p>

Button → User click செய்தால் gowtham() function ஓடும். <p id="timer">10 → Screen-ல number காட்ட ஒரு இடம் வேணும் — அதுதான் இந்த p tag. id="timer" வச்சிருக்கோம் — JavaScript அதை பிடிக்க.

STEP 02

Variable ஏன்?

let time = 10

Timer எங்கிருந்து start ஆகணும்? 10இல் — அதை store பண்ண ஒரு variable வேணும்.

STEP 03

Function ஏன்?

function gowtham() {

Button click ஆனதும் மட்டும் timer ஓடணும் — அதனால function-ல வச்சிருக்கோம். Click இல்லன்னா ஓடாது.

STEP 04

setInterval ஏன்?

let interval = setInterval(() => {
    ...
}, 1000)

setInterval → ஒவ்வொரு 1000ms (1 second)-லயும் உள்ளே இருக்க code-ஐ திரும்ப திரும்ப run பண்ணும். interval variable-ல save பண்றோம் — பிறகு நிறுத்த வேண்டி இருக்கும்.

STEP 05

innerHTML ஏன்?

document.getElementById('timer').innerHTML = time

Screen-ல காட்டணும் — id="timer" இருக்க p tag-ஐ பிடிச்சு அதுல time value போடுறோம்.

STEP 06

time-- ஏன்?

time--

ஒவ்வொரு second-லயும் number ஒன்று குறைய வேண்டும் — 10→9→8... time-- என்பது time = time - 1 மாதிரிதான்.

STEP 07

if Condition ஏன்?

if (0 > time) {
    clearInterval(interval)
    document.getElementById('timer').innerHTML = "Time's Up"
}

time 0-க்கும் கீழே போனதும்: clearInterval(interval) → Timer-ஐ நிறுத்து (இல்லன்னா -1, -2 போகும்). "Time's Up" → Screen-ல காட்டு.

Full Flow Visual
Click Button ↓ gowtham() runs ↓ setInterval starts (every 1 second) ↓ Show time → time-- → check if time < 0? ↓ ↓ NO → repeat YES → Stop + "Time's Up"
02
Event Bubbling

Explain and demonstrate
Event Bubbling

// Event Bubbling என்னன்னா?

ஒரு element-ஐ click செய்தால், அந்த event கீழிருந்து மேலே போகும் — child → parent → document. நீ கல்லை தண்ணீரில் போட்டால் bubble மேலே வரும் — அதுதான் Event Bubbling!

உன் Code Flow — Button click செய்தால்...
button#child ↓ bubble div#paradiv ↓ bubble document

Console Output:

Child Element        ← முதலில்
ParaDiv Clicked      ← இரண்டாவது
body Element Clicked ← மூன்றாவது
HTML

HTML Structure ஏன் இப்படி?

<div id="paradiv">        <!-- Parent -->
    <button id="child">   <!-- Child -->
        Click Me
    </button>
</div>

Button, div-க்கு உள்ளே இருக்கு — அதனால button click = div-உம் technically click ஆச்சு.

LISTENERS

3 Event Listeners ஏன்?

// 1. Button-ல listener
document.getElementById('child').addEventListener("click", function() {
    console.log('Child Element');
})

// 2. Div-ல listener
document.getElementById('paradiv').addEventListener("click", function() {
    console.log('ParaDiv Clicked');
})

// 3. Document-ல listener
document.addEventListener("click", function() {
    console.log('body Element Clicked');
})

மூன்று levels-லயும் வச்சிருக்கோம் — bubble எப்படி ஏறுதுன்னு பார்க்க.

STOP

Bubbling-ஐ நிறுத்த — stopPropagation()

document.getElementById('child').addEventListener("click", function(e) {
    console.log('Child Element');
    e.stopPropagation()  // ← இங்கே நிற்கும், மேலே போகாது
})

Console Output இப்போ: Child Element ← இது மட்டும் வரும்.

USE CASE

Real Life Use Case

// ஒவ்வொரு button-க்கும் தனி listener வேண்டாம்
// Parent-ல ஒரே ஒரு listener போதும்!

document.getElementById('paradiv').addEventListener("click", function(e) {
    if (e.target.id === 'child') {
        console.log('Button clicked!')
    }
    // e.target → யாரை click செய்தோம் அது
})

இதுதான் Event Delegation — Bubbling-ஐ smart-ஆ use பண்றது.

Full Visual Flow
👆 Button Click ↓ [button#child] → "Child Element" ↓ bubble [div#paradiv] → "ParaDiv Clicked" ↓ bubble [document] → "body Element Clicked" stopPropagation() போட்டால்: [button#child] → "Child Element" ✋ STOP — மேலே போகாது
03
Custom map()

Implement a custom map()
without using built-in map

// Question Meaning என்ன?

JavaScript-ல array.map() already இருக்கு — ஆனா அதை use பண்ணாம நாமே உருவாக்கணும். அதாவது map() உள்ளே என்ன நடக்குதுன்னு புரிஞ்சுக்க!

Built-in map() என்ன செய்யும்?
let num = [1, 2, 3, 4, 5, 6]
let result = num.map(function(n) {
    return n * 2
})
// result = [2, 4, 6, 8, 10, 12]
  • Array-ஐ எடுக்கும்
  • ஒவ்வொரு element-க்கும் function run பண்ணும்
  • புதிய array return பண்ணும்
STEP 01

Function Definition

function custommap(arr, callback) {

arr → நாம் pass பண்ற array [1,2,3,4,5,6]. callback → ஒவ்வொரு element-க்கும் என்ன செய்யணும்னு சொல்ற function.

STEP 02

Empty Result Array

let result = []

Original array-ஐ மாத்தக்கூடாது — புதிய array-ல save பண்ணோம்.

STEP 03

Loop

for (let i = 0; i < arr.length; i++) {

Array-ல இருக்க ஒவ்வொரு element-யும் ஒரு முறை தொடு.

STEP 04

callback() Call — Core Logic!

result.push(callback(arr[i], i, arr))
ParameterValueஎன்னன்னா
arr[i]1, 2, 3...Current element
i0, 1, 2...Index number
arr[1,2,3,4,5,6]Full array

callback() run ஆகும் → return value-ஐ result-ல push பண்ணும்.

STEP 05

Return

return result

எல்லா elements-உம் process ஆனதும் புதிய array return.

STEP 06

Usage

let num = [1, 2, 3, 4, 5, 6]

let dub = custommap(num, function(nums) {
    return nums * 2
})

console.log(dub) // [2, 4, 6, 8, 10, 12]
Full Flow Visual
num = [1, 2, 3, 4, 5, 6] ↓ custommap() starts → result = [] ↓ i=0 → callback(1) → 1×2=2 → result = [2] i=1 → callback(2) → 2×2=4 → result = [2,4] i=2 → callback(3) → 3×2=6 → result = [2,4,6] i=3 → callback(4) → 4×2=8 → result = [2,4,6,8] i=4 → callback(5) → 5×2=10 → result = [2,4,6,8,10] i=5 → callback(6) → 6×2=12 → result = [2,4,6,8,10,12] ↓ return [2, 4, 6, 8, 10, 12]
Same Function — Different Uses
// Double
custommap(num, function(n) { return n * 2 })
// [2, 4, 6, 8, 10, 12]

// Square
custommap(num, function(n) { return n * n })
// [1, 4, 9, 16, 25, 36]

// Add index
custommap(num, function(n, i) { return n + i })
// [1, 3, 5, 7, 9, 11]

Same custommap() — callback மாத்தினா result மாறும்!

Flatten Array

Flatten a nested array
without using flat()

// Question Meaning என்ன?

Array-க்கு உள்ளே array இருக்கு — எல்லாத்தையும் ஒரே level-ல கொண்டு வா.

// Input  → [1, [2, 3], [4, [5, 6]]]
// Output → [1, 2, 3, 4, 5, 6]
STEP 01

Input Array

let arr = [1, [2, 3], [4, [5, 6]]]
IndexValueType
01Number
1[2, 3]Array inside Array
2[4, [5, 6]]Array inside Array inside Array
STEP 02

Result Array

let result = []

எல்லா numbers-யும் இங்கே collect பண்ணோம் — outside வச்சிருக்கோம் ஏன்னா function திரும்ப திரும்ப call ஆகும், அப்போவும் data தொலையாம இருக்கணும்.

STEP 03

Function + Loop

function arrs(nums) {
    for (let i = 0; i < nums.length; i++) {

Pass ஆன array-ல ஒவ்வொரு element-யும் check பண்ணு.

STEP 04

The Key Logic — Brain of the program!

if (Array.isArray(nums[i])) {
    arrs(nums[i])         // ← Recursion!
} else {
    result.push(nums[i])  // ← Number இருந்தா save பண்ணு
}
  • Element Array-ஆ? → அதை மீண்டும் arrs()-ல போடு (Recursion)
  • Element Number-ஆ? → result-ல push பண்ணு
Recursion என்னன்னா?

Function தன்னையே திரும்ப call பண்றது.

// arrs([1,[2,3],[4,[5,6]]])  call பண்றோம்
//   └─ [2,3] கண்டால் → arrs([2,3]) call பண்றோம்
//        └─ [5,6] கண்டால் → arrs([5,6]) call பண்றோம்
Full Flow Visual
arrs([1, [2,3], [4,[5,6]]]) │ ├── i=0 → 1 → Number! → result = [1] │ ├── i=1 → [2,3] → Array! → arrs([2,3]) │ │ │ ├── i=0 → 2 → result = [1,2] │ └── i=1 → 3 → result = [1,2,3] │ └── i=2 → [4,[5,6]] → Array! → arrs([4,[5,6]]) │ ├── i=0 → 4 → result = [1,2,3,4] │ └── i=1 → [5,6] → Array! → arrs([5,6]) │ ├── i=0 → 5 → result = [1,2,3,4,5] └── i=1 → 6 → result = [1,2,3,4,5,6] Final → [1, 2, 3, 4, 5, 6] ✅
Why Recursion? — Simple Comparison
// ❌ Normal loop — இது work ஆகாது deeply nested-க்கு
for (let i = 0; i < arr.length; i++) {
    result.push(arr[i])  // [2,3] push ஆகும், உள்ளே போகாது
}

// ✅ Recursion — எத்தனை level இருந்தாலும் உள்ளே போகும்
function arrs(nums) {
    for (...) {
        if (Array.isArray(nums[i])) arrs(nums[i]) // Level எத்தனையானாலும்!
        else result.push(nums[i])
    }
}

Search Filter

Search filter that dynamically
filters table data while typing

// Question Meaning என்ன?

Type பண்ண பண்ண — real time-ல table filter ஆகணும்.

User types "go"  → Gowtham row மட்டும் காட்டு
User types "mo"  → Moni row மட்டும் காட்டு
User types ""    → எல்லாரும் காட்டு
STEP 01

HTML Structure

<input type="text" placeholder="Enter the Values" id="search">

User type பண்ண ஒரு box வேணும் — id="search" வச்சிருக்கோம் JavaScript பிடிக்க.

<table border="1">
    <tr>
        <th>Name</th>     ← Header Row (index 0)
        <th>City</th>
    </tr>
    <tr>
        <td>Gowtham</td>  ← Data Row (index 1)
        <td>Thanjavur</td>
    </tr>
    <tr>
        <td>Moni</td>     ← Data Row (index 2)
        <td>Rameswaram</td>
    </tr>
</table>
STEP 02

Element Grab

let search = document.getElementById('search')

Input box-ஐ JavaScript-ல பிடிச்சு search variable-ல வச்சோம்.

STEP 03

keyup Event ஏன்?

search.addEventListener("keyup", function () {

keyup → User ஒவ்வொரு key விட்ட உடனே function run ஆகும்.
