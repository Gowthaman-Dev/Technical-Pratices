What is an Array?

Arrays in JavaScript are used to store multiple values in a single variable.
An array can store different data types, such as numbers, strings, objects, and even other arrays (multi-dimensional arrays).

Interview line:

An array allows storing and managing multiple values under one variable name.

2. Accessing Array Values

Array values are accessed using index numbers.
Indexes always start from 0. 

const freetimes = ['Mobile using', 'Playing Game', 'RRR', 'KGF'];

console.log(freetimes[0]); // 'Mobile using'
console.log(freetimes[3]); // 'KGF'

Index 0 ‚Üí first element

Index 3 ‚Üí fourth element

Interview line:

JavaScript arrays use zero-based indexing.

3. Arrays of Objects

Arrays often store objects, especially when handling data from APIs or databases.
Arrays often store collections of objects, which is common when working with data (e.g., from an API): . 

const foods = [
  { id: 1, title: 'Chicken Biriyani', dietary: 'non-veg' },
  { id: 2, title: 'Veg Biriyani', dietary: 'veg' },
  { id: 3, title: 'Sambar', dietary: 'veg' }
];

Accessing an object using index

console.log(foods[1]);

This returns the second object in the array.

Accessing a property inside the object

console.log(foods[1].title); // 'Veg Biriyani'

Interview line:

Arrays of objects are commonly used to store structured data like API responses.

4. Iterating Through Arrays (Loops)

Loop Type Explanation (Simple English) Syntax Example
for loop Traditional loop that uses an index number. We manually control the start, end, and increment. Useful when we need full control over iteration. for(let i = 0; i < foods.length; i++) { console.log(foods[i].title); }
for...in loop Iterates over the indexes (keys) of the array. Index is used to access the value. Mostly used for objects, but can also be used with arrays. for (let index in foods) { console.log(foods[index].title); }
for...of loop Iterates over the values (elements) directly. No need to use index. Cleaner and more readable. for (let fooditem of foods) { console.log(fooditem.title); }
forEach() A built-in array method used for iteration. Commonly used to display data in UI. Executes a function for each array element. foods.forEach(food => console.log(food.title))

5. Multi-dimensional Arrays

Explanation (Simple English):

A multi-dimensional array is an array that contains other arrays inside it.
It is used to store data in rows and columns like a table or matrix.

Example:

const matrix = [
  [11, 12, 13],
  [21, 22, 23],
  [31, 32, 33],
  [41, [53, 43, 23], 43]
];

console.log(matrix);

üëâ Here, matrix is a multi-dimensional array because it has arrays inside another array.

üëâ One value ([53, 43, 23]) is nested even deeper.

Flattening Multi-dimensional Arrays

Explanation (Simple English):

The .flat() method is used to convert a multi-dimensional array into a single array.
We can control how deep the array should be flattened using a number.

Method Description Output (for above example)
matrix.flat() Flattens only one level by default [11, 12, 13, 21, 22, 23, 31, 32, 33, 41, [53, 43, 23], 43]
matrix.flat(2) Flattens up to 2 levels (or use Infinity for full flatten) [11, 12, 13, 21, 22, 23, 31, 32, 33, 41, 53, 43, 23, 43]

üëâ If nested array depth is more, increase the flat value.

‚úÖ Interview Line

English:
Multi-dimensional arrays store arrays inside arrays, and the flat() method is used to convert them into a single array.
