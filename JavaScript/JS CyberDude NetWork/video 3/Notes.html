Object destructuring in JavaScript is a useful feature that allows us to extract properties from an object and store them in variables. It makes the code cleaner and easier to read compared to accessing properties one by one.

1. Basic Destructuring

We can extract object properties into variables using the same property names.

Syntax:

const { property1, property2 } = objectName;

Example:

const person = { firstname: "rocky", lastname: "gowtham" };
const { firstname, lastname } = person;

console.log(firstname, lastname); // rocky gowtham

2. Assigning Default Values

If a property does not exist in the object, we can give a default value. This helps to avoid undefined.

Syntax:

const { propertyName = defaultValue } = objectName;

Example:

const { age, favcolor = "red" } = person;
console.log(age, favcolor); // 21 red

3. Renaming Properties (Aliasing)

We can rename a property while destructuring.

Syntax:

const { oldPropertyName: newVariableName } = objectName;

Example:

const { mylover: mypondatti } = person;
console.log(mypondatti); // Jerlin Monisha

4. Destructuring without Declaration

If variables are already declared, we can still use destructuring. In this case, we must wrap the statement in parentheses ().

Syntax:

let var1, var2;
({ var1, var2 } = objectName);

Example:

  favnum1: "2",
  favnum2: "232",
  favnum3: "32",
  favnum4: "43",
  favnum22: "23"
