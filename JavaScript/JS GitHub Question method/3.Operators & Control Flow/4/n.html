JavaScript Operators & Control Flow - Notes

1. Difference Between == and ===

== checks value equality (type conversion allowed).
=== checks value + type equality (strict equality).


console.log(5 == '5');  // true
console.log(5 === '5'); // false

2. Iterating Over Arrays and Objects

Use for...of for arrays and for...in for objects.


// Array
const arr = [10, 20, 30];
for (const val of arr) console.log(val);

// Object
const obj = {a:1, b:2};
for (const key in obj) console.log(key, obj[key]);

3. Break & Continue

break: exit loop, continue: skip current iteration.


for(let i=0; i<5; i++) {
  if(i===3) break;
  if(i===1) continue;
  console.log(i);
}
// Output: 0,2

4. Ternary Operator

Shorthand for if-else: condition ? valueIfTrue : valueIfFalse


const age = 18;
const status = age >= 18 ? 'Adult' : 'Minor';
console.log(status); // Adult

5. Accessing Array Index During Iteration


const arr = ['a','b','c'];
arr.forEach((val, index) => {
  console.log(index, val);
});

6. Switch Statement


const day = 2;
switch(day) {
  case 1: console.log('Mon'); break;
  case 2: console.log('Tue'); break;
  default: console.log('Other'); 
}
// Output: Tue

7. Rest Parameters

Collect multiple arguments into an array.


function sum(...nums) {
  return nums.reduce((a,b)=>a+b,0);
}
console.log(sum(1,2,3)); // 6

8. Spread Operator

Expand array/object into individual elements.


const arr1 = [1,2];
const arr2 = [...arr1,3,4];
console.log(arr2); // [1,2,3,4]

9. Spread vs Rest

Rest: function args (...args)
Spread: expand array/object into elements
