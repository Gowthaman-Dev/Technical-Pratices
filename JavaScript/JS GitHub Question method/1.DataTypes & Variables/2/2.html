How Do You Check the Data Type of a Variable in JavaScript?

Answer

In JavaScript, you can check the data type of a variable using the typeof operator.

Syntax

typeof variable;

JavaScript Data Types Overview

JavaScript is a dynamically typed language, meaning a variable can hold different data types at different times.

1. Primitive Data Types

Primitive types store single, immutable values and are stored by value.

Type Description Example
String Textual data let name = "Gowtham";
Number Integers & decimals (includes NaN, Infinity) let age = 25;
Boolean Logical value let isLoggedIn = true;
Undefined Declared but not assigned let user;
Null Intentional absence of value let data = null;
Symbol Unique identifier (ES6) let id = Symbol("id");
BigInt Large integers (ES2020) let num = 123n;

2. Non-Primitive (Reference) Data Types

Non-primitive types are mutable and stored by reference.

Type Example
Object { name: "Gowtham" }
Array [1, 2, 3]
Function function test() {}
Built-ins Date, Map, Set, RegExp

Using typeof Operator

typeof "Hello";      // "string"
typeof 25;           // "number"
typeof true;         // "boolean"
typeof undefined;    // "undefined"
typeof Symbol();     // "symbol"
typeof 123n;         // "bigint"
typeof {};           // "object"
typeof [];           // "object"
typeof function(){}; // "function"
