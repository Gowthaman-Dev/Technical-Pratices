Technical Vocabulary (English – Tamil)

Useful technical words for programming & interviews

1. Conflicts (Collisions)

Tamil Meaning: மோதல்கள் (பெயர் மோதல்கள் / இட மோதல்கள்)

Pronunciation: கான்-ஃப்ளிக்ட்ஸ் (கொலிஷன்ஸ்)

Sentence: Conflicts (collisions) occur when two or more things clash with each other.

2. Instead of quotes

Tamil Meaning: quotes க்கு பதிலாக

Pronunciation: இன்-ஸ்டெட் ஆஃப் கோட்ஸ்

3. Embedded

Tamil Meaning: உட்பொதிக்கப்பட்ட / உள்ளே இணைக்கப்பட்ட

Pronunciation: எம்-பெட்-இட்

Explanation: Something is embedded when it is fixed or placed inside another thing.

4. Interpolation

Tamil Meaning: இடைச்செருகல் / மாறிகளை நடுவில் சேர்த்தல்

Pronunciation: இன்-டர்-போ-லே-ஷன்

Explanation: Interpolation means inserting variables directly inside a string.

5. Concatenation

Tamil Meaning: இணைத்தல் / சரங்களை சேர்த்தல்

Pronunciation: கான்-காட்-டினே-ஷன்

Explanation: Concatenation means joining multiple strings together.

6. Malicious

Tamil Meaning: தீங்கிழைக்கும் / தீய நோக்கத்துடன் செய்யப்படும்

Pronunciation: ம-லி-ஷஸ் (Malishus)

Explanation: Malicious means done to cause harm intentionally.

7. Causing

Meaning: காரணமாக இருப்பது

Pronunciation: காஸிங்

Example: Global variables are causing bugs.

8. Predictable

Meaning: முன்னதாகவே கணிக்கக்கூடியது

Pronunciation: ப்ரிடிக்டபிள்

Example: Clean code is predictable.

9. Unintended

Meaning: திட்டமிடாத / தவறுதலான

Pronunciation: அன்-இன்டெண்டட்

Example: Unintended changes cause bugs.

10. Garbage

Meaning: தேவையில்லாத data / குப்பை

Pronunciation: கார்பேஜ்

Example: Unused objects become garbage.

11. Modularity

Meaning: code-ஐ தனித்தனி பகுதிகளாக பிரிப்பது

Pronunciation: மாட்யூலாரிட்டி

Example: Modularity improves code reuse.

12. Isolated

Meaning: தனியாக பிரிக்கப்பட்ட

Pronunciation: ஐஸலேட்டெட்

Example: Each function should be isolated.

13. Stability

Meaning: நிலைத்தன்மை

Pronunciation: ஸ்டெபிலிட்டி

Example: Less global variables increase stability.

14. Pollution

Meaning: மாசுபடுத்தல்

Pronunciation: பொல்யூஷன்

Example: Global scope pollution is dangerous.

15. Collisions

Meaning: மோதல் / பெயர் மோதல்

Pronunciation: கொலிஷன்ஸ்

Example: Variable name collisions cause errors.

Final Interview Tip

Understanding technical vocabulary improves communication, code clarity, and interview confidence.
