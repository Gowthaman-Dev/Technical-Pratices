6. Code Examples (Template Literals)

let name = "Rocky";
let age = 23;

// Using template literals
let message = `My name is ${name} and I am ${age} years old`;

console.log(message);

// Output:
// My name is Rocky and I am 23 years old

тЬЕ Multi-line String Example

let text = `Hello,
Welcome to JavaScript!`;

console.log(text);

Template literals are used to create readable strings in JavaScript using backticks and allow inserting variables with ${ }.

Avoiding + (Concatenation)

const firstName = "Rocky";
const lastName = "Rocks";

// Without template literals (using +)
const fullName1 = firstName + " " + lastName;
console.log(fullName1); // Rocky Rocks

// With template literals (no + needed)
const fullName2 = `${firstName} ${lastName}`;
console.log(fullName2); // Rocky Rocks
