How Do You Convert a String to a Number in JavaScript?

In JavaScript, a string can be converted to a number using Number(), parseInt(), parseFloat(), or the unary plus (+) operator.

Common Ways to Convert

1. Number()

  • Converts the entire string to a number
  • Returns NaN if the string is not fully numeric
Number("123");     // 123
Number("123.45");  // 123.45
Number("123px");   // NaN

2. Unary Plus (+)

  • Works the same way as Number()
  • Short and fast syntax
+"42";    // 42
+"42px";  // NaN

3. parseInt()

  • Extracts an integer from the beginning of the string
  • Stops parsing when it encounters a non-numeric character
parseInt("100");     // 100
parseInt("100.5");   // 100
parseInt("100px");   // 100

4. parseFloat()

  • Extracts a number including decimal values
  • Stops parsing at the first invalid character
parseFloat("99.99");   // 99.99
parseFloat("99.99px"); // 99.99
