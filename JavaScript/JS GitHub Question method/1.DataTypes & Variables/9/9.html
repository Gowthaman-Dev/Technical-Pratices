What Is the Spread Operator and How Is It Used?

Answer

The spread operator (...) is a JavaScript feature that allows you to expand elements of an array, object, or other iterables into individual values.

It is commonly used to copy arrays or objects, merge multiple arrays or objects, and pass array elements as arguments to functions, making code cleaner and more readable.

Example (If Asked)

const arr = [1, 2, 3];
const copy = [...arr];

Using Spread with Functions

The spread operator can be used to pass array values as individual arguments to a function. 

Math.max(...[1, 5, 3]); // 5
