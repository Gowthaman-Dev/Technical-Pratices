тЬЕ 4. Map with Object / Array value

Option 1: value as Object

let map = new Map([
  ['name', { value: 'gowtham', extra: 'as' }],
  ['age', { value: 22, extra: 'as' }]
]);

for (let [key, val] of map) {
  console.log(key, val.value, val.extra);
}

Output:

name gowtham as
age 22 as

Option 2: value as Array

let map = new Map([
  ['name', ['gowtham', 'as']],
  ['age', [22, 'as']]
]);

for (let [key, [value, asd]] of map) {
  console.log(key, value, asd);
}

Output:

name gowtham as
age 22 as

Interview One-liner: A Map stores data as keyтАУvalue pairs only. If we need more data, we store it inside the value as an object or array.
