Various Ways to Create Objects in JavaScript

Interview-ready Answer (Simple English)

In JavaScript, objects can be created in multiple ways. The most commonly used methods are Object Literals, Constructor Functions, and ES6 Classes. Other methods like Object() and Object.create() are less commonly used but important to know.

1. Object Literal {} ‚≠ê (MOST IMPORTANT)

Definition

The simplest and most common way to create an object using key-value pairs.

Example

  name: "Alice",
  age: 30
};

2. Object() Constructor

Definition

Creates an object using the built-in Object constructor and then adds properties.

Example

const car = new Object();
car.make = "Toyota";
car.model = "Camry";
Interview Points:
  • Older style
  • Rarely used in real projects
  • üëâ Know the concept only

3. Object.create() (Prototype-based)

Definition

Creates a new object using another object as its prototype.

Example

const proto = { greeting: "Hello" };
const user = Object.create(proto);
user.name = "Bob";
Interview Points:
  • Used for prototypal inheritance
  • Theory-oriented topic
  • üëâ Basic understanding is enough

4. Constructor Function ‚≠ê‚≠ê (IMPORTANT)

Definition

A function used as a blueprint to create multiple objects using the new keyword.

Example

function Person(name, age) {
  this.name = name;
  this.age = age;
}

const person1 = new Person("Charlie", 25);
Interview Points:
  • Used before ES6
  • Creates multiple similar objects
  • üëâ Fresher SHOULD study

5. ES6 Classes ‚≠ê‚≠ê‚≠ê (VERY IMPORTANT)

Definition

A modern and cleaner way to create objects using class and constructor.

Example

class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(this.name + " says Woof!");
  }
}

const myDog = new Dog("Buddy");
Interview Points:
  • Most used in modern JavaScript
  • Easy to read and maintain
  • üëâ Fresher MUST study
