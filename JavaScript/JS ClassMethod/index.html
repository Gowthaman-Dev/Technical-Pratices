JavaScript Interview Questions & Answers

1. What is JavaScript?

Answer:
JavaScript is a high-level, interpreted programming language used to make web pages interactive.

One Line Definition

JavaScript (JS) is a high-level programming language mainly used for interactive web applications.

Difference between JS and Other Languages

  • JavaScript runs mostly in the browser (client-side).
  • It is loosely typed.
  • Supports event-driven and functional programming.

2. Variables

Difference between var, let, const

Keyword Scope Reassign Redeclare
var Function Yes Yes
let Block Yes No
const Block No No

Can you reassign a const variable?

No. But object/array values inside const can be modified. 

const arr = [1,2,3];
arr.push(4); // Works

3. Keywords

Some Reserved Keywords

break, case, catch, class, const, continue, for, function, if, let, return, switch, this, try, typeof, var, while

Can we use keywords as variable names?

No. Keywords are reserved. 

var let = 10; // Error

Rules for Variable Names

  • Can contain letters, digits, _ and $
  • Cannot start with digit
  • Cannot use keywords
  • Case-sensitive

4. Declaration Phase (Memory Creation)

Before execution, JS engine allocates memory for variables and functions.

  • var → initialized with undefined
  • let & const → created but not initialized

Hoisting

console.log(a); // undefined
var a = 5;

console.log(b); // ReferenceError
let b = 10;

5. Execution Phase

Code runs line by line and values are assigned.

Declaration Phase Execution Phase
Memory allocation Value assignment
