8️⃣ break & continue (Loop control)

📌 Tamil

break → loop-ah full-aa stop

continue → current step skip pannitu next iteration

for(let i = 1; i <= 5; i++){ if(i === 3) break; console.log(i); }

🌍 English
Break stops the loop completely; continue skips the current iteration.

👉 Prime number logic-la break romba important.

9️⃣ Loop inside condition (if inside for)

📌 Tamil

For loop mattum illa

for + if combine panninaa thaan real logic varum

for(let i = 1; i <= 10; i++){ if(i % 2 === 0){ console.log(i); } }

🌍 English
Most loop logic is built using conditions inside loops.
