1️⃣ “User register பண்ணினா அந்த data எங்க save ஆகும்?”

👉 Students answer: Database
👉 அப்போ சொல்லு: Backend server database connect பண்ணி data store பண்ணும்.
2️⃣ Database Concept (தமிழ்)
Database = Application data permanent-ஆ save ஆகும் place. Example data: name, email, password. Express server database connect பண்ணி data save / fetch / update / delete பண்ணும்.
“Express allows backend applications to connect with databases and manage data.”
3️⃣ Popular Databases
MongoDB MySQL
MongoDB connect பண்ண usually use பண்ணுற package: Mongoose
4️⃣ Database Connection Flow
Frontend⬇️Express Server⬇️Database⬇️Response
1️⃣ User request அனுப்புவார்   2️⃣ Express database கிட்ட query அனுப்பும்   3️⃣ Database data return பண்ணும்
5️⃣ CRUD Operations
Create new data save
Read data fetch
Update data modify
Delete data remove
Create → register user   Read → get users   Update → update profile   Delete → delete account
6️⃣ Express + MongoDB Example
import express from "express" import mongoose from "mongoose" const app = express() mongoose.connect("mongodb://localhost:27017/mydb") app.get("/users", (req, res) => { res.send("Users fetched from database") }) app.listen(5000)
👉 Express server create · database connect · route மூலம் data access
7️⃣ Real Project Example
POST /register GET /users PUT /updateUser DELETE /deleteUser
👉 இந்த APIs database CRUD operations perform பண்ணும்.
8️⃣ Class Demo (Important)

mongoose.connect("mongodb://localhost:27017/mydb") .then(()=>console.log("Database connected"))

9️⃣ Final Summary (Board)
Board ல எழுதுங்க: Client → Express Server → Database → CRUD Operations → Response
