Express.js Full Concept Interview Points

1️⃣ Server create panna
Tamil: Node.js ல server create பண்ண Express use பண்ணுறோம். HTTP request handle பண்ண easy methods இருக்கு.
👉 “Express is a Node.js framework used to easily create servers and handle HTTP requests.”
2️⃣ Routing handle panna route
Tamil: Frontend இருந்து வரும் URL request-களை handle பண்ண routes create பண்ணுறோம்.
Example
app.get("/users",(req,res)=>{
res.send("Users data")
})
👉 “Express helps to manage routing like GET, POST, PUT and DELETE requests.”
3️⃣ Middleware support mw
Tamil: Request server கு வரும்போது அதை process பண்ண middleware use பண்ணுறோம்.
Example:
authentication
logging
validation
app.use(cors())
👉 “Express supports middleware to process requests before sending a response.”
4️⃣ REST API create panna API
Tamil: Frontend மற்றும் backend communicate பண்ண API create பண்ண Express use பண்ணுறோம்.
app.post("/register",registerUser)
👉 “Express is commonly used to build RESTful APIs for web applications.”
5️⃣ Fast and Lightweight ⚡
Tamil: Express framework lightweight ஆக இருப்பதால் fast performance கிடைக்கும்.
👉 “Express is a fast and lightweight framework for building backend applications.”
6️⃣ Database connect panna DB
Tamil: Express மூலம் MongoDB / MySQL connect பண்ணி CRUD operations செய்யலாம்.
Example:
Create
Read
Update
Delete
👉 “Express makes it easy to connect databases and perform CRUD operations.”
7️⃣ Request and Response Object req/res
Tamil: Client இருந்து வரும் data req (request) object-ல இருக்கும். Server response res (response) object மூலம் send பண்ணுவோம்.
Example
app.get("/",(req,res)=>{
res.send("Hello World")
})
👉 “Express uses request and response objects to handle client-server communication.”
8️⃣ Static Files serve panna static
Tamil: Images, CSS, JS files serve பண்ண Express use பண்ணலாம்.
Example
app.use(express.static("uploads"))
👉 “Express can serve static files like images, CSS and JavaScript.”
9️⃣ Error Handling error
Tamil: Application error handle பண்ண middleware use பண்ணலாம்.
Example:
app.use((err,req,res,next)=>{
res.status(500).send("Server Error")
})
👉 “Express provides middleware to handle application errors.”
