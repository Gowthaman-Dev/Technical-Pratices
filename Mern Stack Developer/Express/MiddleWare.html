1️⃣ “User login பண்ணாமல் dashboard open பண்ண try பண்ணினா என்ன ஆகும்?”

👉 Students answer: Login page redirect ஆகும்
👉 Server first check பண்ணும் login இருக்கா இல்லையா · அந்த checking process தான் Middleware
2️⃣ Middleware Definition (தமிழ்)
Middleware என்றால் என்ன? Request server கு வரும்போது response send பண்ணும் முன்னாடி extra process run ஆகும் function தான் middleware.
“Middleware is a function that processes a request before sending the response.”
3️⃣ Request Flow Diagram (board draw)
Client ⬇️ Middleware ⬇️ Route ⬇️ Response
👉 Request first middleware கு வரும் · middleware check பண்ணும் · next route கு அனுப்பும்

5️⃣ Basic Middleware Example
import express from "express" const app = express() app.use((req, res, next) => { console.log("Middleware executed") next() }) app.get("/users", (req, res) => { res.send("Users data") }) app.listen(5000)
👉 app.use() → middleware    👉 next() → next function call
6️⃣ Middleware Without next() ⚠️ important
If next() call பண்ணலனா request stop ஆகும். Route run ஆகாது.
app.use((req, res, next) => { console.log("Stop here") // next() ❌ not called }) // route below never runs app.get("/users", ...)
