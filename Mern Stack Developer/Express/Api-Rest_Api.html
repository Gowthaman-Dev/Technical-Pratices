API என்றால் என்ன? Frontend இருந்து வரும் request-ஐ backend handle பண்ணி data send பண்ணும் communication system தான் API.

4️⃣ What is REST API ?
REST API = Standard way to create APIs using HTTP methods
GET data fetch
POST create data
PUT update data
DELETE delete data
5️⃣ Express REST API Example
import express from "express" const app = express() app.post("/register", (req, res) => { res.send("User Registered") }) app.listen(5000)
👉 /register → API endpoint    POST → create data
