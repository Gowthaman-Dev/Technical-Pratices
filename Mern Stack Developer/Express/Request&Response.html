1️⃣ “Browser server கிட்ட data அனுப்பினா அந்த data எங்க வரும்?”

👉 Client அனுப்பும் data → Request    👉 Server திருப்பி அனுப்பும் data → Response
✨ Explain பண்ணு: Client → Request, Server → Response
2️⃣ Request / Response Definition
Request (req) Client இருந்து server கு வரும் information. Example: form data, URL parameters, query data.
Response (res) Server இருந்து client கு அனுப்பும் reply / result.
“Express uses request and response objects to handle client-server communication.”
3️⃣ Communication Flow (Board draw)
Client⬇️Request (req)⬇️Express Server⬇️Response (res)⬇️Client
👉 Browser request அனுப்பும் · Express process பண்ணும் · Response send பண்ணும்
4️⃣ Basic Express Example
import express from "express" const app = express() app.get("/", (req, res) => { res.send("Hello World") }) app.listen(5000)
👉 req → request object    res → response object
5️⃣ Request Object Examples
req.body
form data
req.params
URL parameter
req.query
query string
app.get("/users/:id", (req, res) => { console.log(req.params.id) })
👉 Request object ல data இருக்கும்.
6️⃣ Response Object Examples
res.send() → text res.json() → JSON res.status() → code
res.json({ name: "Gowtham" })
👉 Response object client கு data send பண்ணும்.

Client → req → Server → res → Client
🔥 “Request object contains data sent by the client, and the response object is used by the server to send data back to the client.”
