4️⃣ Basic Express Routing Example code
import express from "express"
const app = express()

app.get("/users", (req, res) => {
  res.send("Users data")
})

app.listen(5000)
🔹 app.get() → GET request handle   🔹 /users → route URL   🔹 req → request   🔹 res → response

6️⃣ Different HTTP Routes methods
GET Data get
POST Data create
PUT Update
DELETE Delete
app.get("/users")   // GET
app.post("/users")  // POST
app.put("/users")   // PUT
app.delete("/users") // DELETE
