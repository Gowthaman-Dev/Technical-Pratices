TOPIC 05

Lightweight & Fast

🏍️
1️⃣ Lightweight என்னன்னா?
"Lightweight = Unnecessary features இல்லாம, minimum code போதும், fast performance."
அதிக code இல்லாம → Simple → Fast = Express.js ✅
2️⃣ Real Life Example 🚗
Vehicle Comparison
🚛 Heavy TruckSlow
🏍️ BikeFast
Framework Comparison
Heavy FrameworkSlow
Express.jsFast & Lightweight
✅ Express = Bike மாதிரி — lightweight, fast, flexible ✅
3️⃣ Minimum Code — Server Ready
server.js 
import express from "express"

const app = express()

app.get("/", (req, res) => {
  res.send("Hello")
})

app.listen(5000)
🔥 வெறும் 5 lines — Server ready!
Other heavy frameworks இதே காரியத்துக்கு 20+ lines எழுதணும்.
4️⃣ Express Fast ஆ இருக்க 3 காரணங்கள்
1 Minimal Framework — தேவையானது மட்டும் இருக்கும்
2 Node.js Async — Non-blocking architecture → Fast
3 Only Required Features — Extra load இல்லை
5️⃣ Request Flow
Client Request
Express Server← Lightweight, fast handle
Fast Response
6️⃣ Practical Point 📦
தேவையானது மட்டும் add பண்ணலாம் 
import cors from "cors"
import mongoose from "mongoose"

app.use(cors())
✅ தேவையான packages மட்டும் → Application heavy ஆகாது → Fast ✅
7️⃣ Interview Answer (வாயால சொல்லு)

"Express.js is a fast and lightweight framework because it provides only minimal features and is built on top of Node.js which uses non-blocking asynchronous architecture. We can create a server with just a few lines of code and add only the packages we need, keeping the application light and fast."

5️⃣ Express + MongoDB Code
db.js 
import express from "express"
import mongoose from "mongoose"

const app = express()

// Database Connect
mongoose.connect("mongodb://localhost:27017/mydb")
  .then(() => console.log("Database connected")) // ✅ Terminal ல தெரியும்

app.get("/users", (req, res) => {
  res.send("Users fetched from database")
})

app.listen(5000)

.then()Connect success confirm
