1️⃣ “Website-ல images, CSS styles, JavaScript files எங்கிருந்து load ஆகுது?”

👉 Server அந்த files-ஐ browser கு send பண்ணும் · அந்த files தான் Static Files
✨ Explain: Server → static files → browser
2️⃣ Static Files Definition (தமிழ்)
Static files என்றால் என்ன? Server side processing இல்லாமல் direct-ஆ browser கு send பண்ணப்படும் files. Examples: images, CSS files, JavaScript files.
“Static files are files served directly to the client without server processing.”
3️⃣ Example Files (folder structure)
📁 project
📄 server.js
📁 uploads
🖼️ image1.png
🎨 style.css
📜 script.js
இந்த files browser கு serve பண்ண Express use பண்ணலாம்.
4️⃣ Express Static Middleware
import express from "express" const app = express() app.use(express.static("uploads")) app.listen(5000)
👉 express.static() → static files serve பண்ணும் middleware · "uploads" → files இருக்கும் folder
5️⃣ Browser Example
🌐 http://localhost:5000/image1.png
👉 Browser அந்த image-ஐ direct-ஆ show பண்ணும். Express server அந்த file-ஐ client கு send பண்ணுது.
6️⃣ Real Project Example
profile image (🖼️)
CSS styling (🎨)

<img src="http://localhost:5000/profile.png" />
👉 image show பண்ண இப்படி use பண்ணலாம்.
7️⃣ Board Diagram (Teaching)
Browser⬇️Request image⬇️Express Server⬇️Static Folder⬇️Image Response
👉 Board ல draw பண்ணு இந்த flow.
8️⃣ Practical Demo (Important)
1️⃣ uploads folder create 2️⃣ image add பண்ணு 3️⃣ browser ல open பண்ணு
👉 Students immediate concept புரிஞ்சிடுவாங்க.
9️⃣ Final Summary (Board)
Client → Request static file → Express → Folder → Response
🔥 “Express can serve static files like images, CSS and JavaScript using the express.static middleware.”
