⚙️ Middleware முழு விளக்கம் · interview தமிழ்

request ↔ response நடுவில் run ஆகும் function
1️⃣ Middleware என்றால் என்ன?
Tamil: Middleware என்பது request மற்றும் response நடுவில் run ஆகும் function.
Flow: Client Request → Middleware → Route → Response
Frontend இருந்து request வந்ததும் route க்கு போகும் முன் ஒரு function run ஆகும். அதுதான் middleware.
Client Request ➡️ Middleware ➡️ Route ➡️ Response
“Middleware is a function that runs between the request and the response in an Express application.”
2️⃣ Middleware எப்படி வேலை செய்கிறது?
app.use((req, res, next) => { console.log("Request received") next() })
Part   Meaning
req   → client request data
res   → response send பண்ண
next() → next function / route க்கு போக
⚠️ next() call பண்ணலனா request stop ஆகும்
“Middleware functions have access to request, response, and the next function.”
3️⃣ Middleware ஏன் use பண்ணுறோம்?
🔐 Authentication
📋 Logging
✅ Validation
🛡️ Security
1. Authentication: User login இருக்கானா check.
const auth = (req,res,next) => { if(req.headers.token) next() else res.send("Unauthorized") }
2. Logging: Server-க்கு வரும் request track.
app.use((req,res,next) => { console.log(req.method, req.url) next() })
3. Validation: User data check.
if(!req.body.email) return res.send("Email required")
4. Security: JWT verification, API key check.
4️⃣ Middleware types (interview முக்கியம்)
📌 Application Middleware app.use(cors())
📍 Route Middleware app.get("/dashboard", auth, ...)
📦 Third-party Middleware cors, multer, morgan
⚙️ Built-in Middleware express.json()
Example third-party: import cors from "cors"; app.use(cors())
Built-in: app.use(express.json()) → JSON data read பண்ண.
5️⃣ Middleware flow example
app.use((req,res,next) => { console.log("Middleware 1") next() }) app.use((req,res,next) => { console.log("Middleware 2") next() }) app.get("/", (req,res) => { res.send("Hello") })
Output: Middleware 1, Middleware 2, Hello
6️⃣ Real MERN example (Interview level)
const verifyUser = (req,res,next) => { const token = req.headers.authorization if(!token) { return res.status(401).json({msg:"No token"}) } next() } app.get("/dashboard", verifyUser, (req,res) => { res.send("Dashboard data") })
👉 Login இல்லாத user dashboard access பண்ண முடியாது.
7️⃣ Short Interview Answer
“Middleware in Express is a function that runs before the route handler. It is used for authentication, logging, validation, and modifying requests or responses.”
