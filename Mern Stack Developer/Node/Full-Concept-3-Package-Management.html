NPM (Node Package Manager)

Tamil Explanation

NPM என்பது Node.js packages install மற்றும் manage பண்ண பயன்படும் tool.


Node.js install பண்ணும்போது NPM automatic-ஆ install ஆகும்.

NPM use பண்ணி நாம்:

  • External libraries install பண்ணலாம்
  • Dependencies manage பண்ணலாம்
  • Scripts run பண்ணலாம்
  • Packages publish பண்ணலாம்

Example (Package Install)

terminal 
npm install express

இதன் மூலம்:

  • Express package install ஆகும்
  • node_modules folder create ஆகும்
  • package.json update ஆகும்

Important NPM Commands

1️⃣

Install Package

terminal 
npm install express
2️⃣

Install Dev Dependency

Development time மட்டும் use பண்ணும் package.

Example

terminal 
npm install nodemon --save-dev

Examples

nodemon eslint prettier
3️⃣

Install Global Package

System முழுவதும் use பண்ண.

terminal 
npm install -g nodemon
4️⃣

Remove Package

terminal 
npm uninstall express
5️⃣

Install All Dependencies

Project clone பண்ணும்போது use பண்ணுவோம்.

terminal 
npm install
📁 node_modules Folder

Package install பண்ணும்போது node_modules folder create ஆகும்.

இதில் இருக்கும்

  • Installed packages
  • dependencies of dependencies
⭐ English Sentence (Interview)
"NPM is the default package manager for Node.js used to install, manage, and update project dependencies."

package.json

Tamil Explanation

package.json என்பது Node.js project-இன் configuration file.

இதில் இருக்கும்:

  • project information
  • dependencies
  • scripts
  • version

Example

package.json 
{
  "name": "mern-project",
  "version": "1.0.0",
  "description": "Job portal backend",
  "main": "server.js",
  "scripts": {
    "start": "node server.js"
  },
  "dependencies": {
    "express": "^4.18.2"
  }
}

Important Fields

FieldPurpose
nameproject name
versionproject version
descriptionproject details
mainentry file
scriptscommands run பண்ண
dependenciesrequired packages
devDependenciesdevelopment packages

scripts Example

package.json 
"scripts": {
  "start": "node server.js",
  "dev": "nodemon server.js"
}

Run command:

terminal 
npm run dev

dependencies vs devDependencies

dependencies

Production-ல run ஆக தேவையான packages.

Example

  • express
  • mongoose
  • bcrypt
  • jsonwebtoken

devDependencies

Development time மட்டும் use பண்ணும்.

Example

  • nodemon
  • eslint

package.json Create Command

terminal 
npm init
terminal 
npm init -y

இதனால் automatically package.json create ஆகும்.

⭐ English Sentence (Interview)
"package.json is a configuration file that stores project metadata, dependencies, and scripts for a Node.js application."
