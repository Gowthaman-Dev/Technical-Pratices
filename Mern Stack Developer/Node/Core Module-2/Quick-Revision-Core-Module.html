Modules

Node.js Modules

Definition
Module = Reusable JavaScript file

Module என்றால் reusable code கொண்ட JavaScript file. Node.js-ல் code-ஐ small files ஆக split பண்ணி organize பண்ணலாம். Code clean ஆக இருக்கும், reuse பண்ண முடியும், maintain easy ஆகும்.

Example modules: math.js   auth.js   database.js — ஒவ்வொரு file-மும் ஒரு module.

2 Types of Modules
1. Core Modules — Built-in
fs → file system
http → server create
path → file paths
os → operating system info
core.js
const fs = require("fs")  // require() use பண்ணி import
2. User Defined Modules — நாமே create பண்ணும்
math.js
function add(a, b) {
  return a + b
}

module.exports = add
// function export
app.js
const add = require("./math")

console.log(add(2, 3))
// require() import
Output: 5

Global

Global Objects

Definition
Global Objects — No import needed

Node.js application முழுவதும் access பண்ணக்கூடிய objects. Import தேவையில்லை — directly use பண்ணலாம். Node.js program எங்கும் direct-ஆ access பண்ணலாம்.

console process setTimeout setInterval __dirname __filename
6 Important Global Objects
console
Log messages print பண்ண
console.js
console.log("Hello Node")
Output: Hello Node
process
Current Node.js process info
process.js
console.log(process.platform)
Output: win32
__dirname
Current folder path
dirname.js
console.log(__dirname)
Output: C:\project\app
__filename
Current file path
filename.js
console.log(__filename)
Output: full file path
setTimeout
Code later run பண்ண
timeout.js
setTimeout(() => {...}, 2000)
2 seconds பிறகு run ஆகும்
setInterval
Repeated execution
interval.js
setInterval(() => {...}, 1000)
Every second run ஆகும்
Code Examples
dirname.js
console.log(__dirname)
// project folder path
Output:project folder path
process.js
console.log(process.platform)
// win32 or linux
Output:win32

Global vs Non-Global — Quick Trick
Requires import
const fs = require("fs")
fs.readFile(...)

fs — import செய்யணும். Global இல்லை.

Global — no import
console.log("Hello")
// no require() needed

console — import இல்லாம use பண்ணலாம். Global object.

Interview Sentence — Memorize

"Global objects in Node.js are available throughout the application without requiring import."

Process

Process Object

Definition
process — Global Object

Node.js application run ஆகும் current process information கொடுக்கும் global object. Import தேவையில்லை — directly use பண்ணலாம். Environment variables access, command line arguments, process control இதன் மூலம் பண்ணலாம்.

console.log(process)  // no require() needed
4 Important Properties
process.env
Environment variables access பண்ண. .env file values read பண்ணலாம்.
env.js
console.log(process.env.PORT)
Output: 5000
process.argv
Command line arguments. Program run-ல் pass பண்ணும் values store ஆகும்.
argv.js
console.log(process.argv)
Output: ['node', 'app.js', 'hello']
process.exit()
Program immediately stop பண்ண.
exit.js
process.exit()
Program stop ஆகும்
process.pid
Current running process ID.
pid.js
console.log(process.pid)
Output: 2345
process.env — .env File Example
.env file
PORT=5000
env.js
console.log(process.env.PORT)
// Output: 5000

process.env-ல் PORT, DB_URL, JWT_SECRET போன்ற .env file values read ஆகும். MERN project-ல் மிக முக்கியம்.

process.argv — Command Line Arguments Deep Dive
// Terminal command:
// node app.js hello

console.log(process.argv)
// ['C:\nodejs\node.exe', 'C:\project\app.js', 'hello']
IndexValueMeaning
0node pathNode.js location
1file nameRun பண்ணும் file
2helloநாம் pass பண்ணிய argument
Real example — app.js
// node app.js Rocky
console.log("Hello", process.argv[2])
// Output: Hello Rocky

Trick Interview Q — Only user arguments எப்படி get பண்றது?

process.argv[0] = node path, [1] = file name — user arguments [2] from start.
Answer: process.argv.slice(2) → user arguments மட்டும் return ஆகும்.

Final Combined Answer (Modules + Global + Process)

"Node.js uses modules to organize code into reusable files. Global objects are available throughout the application without importing them. The process object provides information about the currently running Node.js process, including environment variables and command-line arguments."

Interview Sentence — Memorize

"The process object provides information and control over the current Node.js process."
