IndexMeaning

4️⃣ Process ID

Deep Dive – Command Line Arguments (process.argv)
1️⃣

First Idea

நாம் Node.js program run பண்ணும்போது extra values pass பண்ணலாம்.

Example command

terminal 
node app.js hello

இதில்

  • node → Node.js run
  • app.js → program file
  • hello → argument

இந்த values எல்லாம் process.argv-ல store ஆகும்.

2️⃣

Code Example

app.js

app.js 
console.log(process.argv)

Program run பண்ணு:

terminal 
node app.js hello
▶ Output
[
  'C:\\Program Files\\nodejs\\node.exe',
  'C:\\project\\app.js',
  'hello'
]

Output Meaning

1file namerun பண்ணும் file
2helloநாம pass பண்ணிய argument
