  • Firefox

02

Step 2 – Problem Create பண்ணணும்

அடுத்த question:

அப்போ சொல்லணும்:

03

Simple definition

👉

Node.js என்பது JavaScript runtime environment.

Tamil meaning

👉

Browser இல்லாமல் JavaScript run பண்ண உதவும் environment.

06

Code Example (Most Important)

பிறகு code show பண்ணணும்.

server.js 
const http = require("http");

const server = http.createServer((req, res) => {
  res.write("Hello from Node.js");
  res.end();
});

server.listen(5000, () => {
  console.log("Server running on port 5000");
});

Explain:

Line 1

const http = require("http");

Node.js built-in http module import பண்ணுறோம்.

Line 2

http.createServer()

Server create பண்ணுறோம்.

Request வந்தால்

res.write("Hello from Node.js")

Client-க்கு response send பண்ணுறோம்.

Port start

server.listen(5000)

Server port 5000-ல் run ஆகும்.

Interview Sentence

Class end-ல சொல்லணும்:

👉

Students memorize this line

English:

⭐ Interview Sentence
"Node.js is a runtime environment that allows JavaScript to run on the server side using the V8 engine."
⭐ Super Teaching Trick

Class-ல இந்த question கேளு:

"If JavaScript runs in browser, why do we need Node.js?"

Answer:

To run JavaScript on the server side.

Node.js Features
1️⃣

Open Source

🗣️

Class-ல சொல்ல: "Node.js open source technology."

Meaning

  • யாரும் free-ஆ download பண்ணி use பண்ணலாம்
  • Developers code contribute பண்ணலாம்

Example

GitHub-ல Node.js code available.

2️⃣

Cross Platform

🗣️

சொல்ல: "Node.js cross-platform."

Meaning

  • Windows
  • Linux
  • Mac

இந்த எல்லா operating system-லவும் run ஆகும்.

Example

ஒரு Node.js application Windows-ல build பண்ணினாலும் Linux server-ல run பண்ணலாம்.

3️⃣

Event-Driven

🗣️

சொல்ல: "Node.js works using event-driven architecture."

Meaning

  • Event நடந்தால் code execute ஆகும்

Example

User click User login API request

இந்த events வந்தால் server respond பண்ணும்.

4️⃣

Non-Blocking I/O

🗣️

சொல்ல: "Node.js uses non-blocking I/O."

Meaning

  • ஒரு request process ஆகும் போது server other requests-ஐ wait பண்ணாது
  • அது multiple requests handle பண்ணும்.
  • இதனால் performance அதிகம்.
5️⃣

Fast Execution (V8 Engine)

🗣️

சொல்ல: "Node.js uses the V8 JavaScript engine."

Meaning

  • V8 engine JavaScript-ஐ machine code-ஆ convert பண்ணும்
  • அதனால் execution speed அதிகம்.

Example

Chrome browser கூட V8 engine தான் use பண்ணுது.

⭐

Interview-ல சொல்லும் sentence

"Node.js features include open source, cross-platform support, event-driven architecture, non-blocking I/O, and fast execution using the V8 engine."
