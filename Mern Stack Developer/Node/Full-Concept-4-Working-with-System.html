File System (fs)

Tamil Explanation

File System (fs) என்பது Node.js built-in module.


இதன் மூலம் files மற்றும் folders-ஐ create, read, write, delete பண்ணலாம்.


Node.js server-ல file operations செய்ய இதை use பண்ணுவோம்.

Examples

  • file read பண்ண
  • file write பண்ண
  • file delete பண்ண
  • folder create பண்ண

Import fs Module

import 
const fs = require("fs")

Important File System Methods

1️⃣

Read File

File content read பண்ண use பண்ணுவோம்.

readFile.js 
const fs = require("fs")

fs.readFile("data.txt", "utf8", (err, data) => {
  if (err) throw err
  console.log(data)
})

👉 File content console-ல print ஆகும்.

2️⃣

Write File

New file create அல்லது file-ல data write பண்ண.

writeFile.js 
fs.writeFile("data.txt", "Hello Node.js", (err) => {
  if (err) throw err
  console.log("File created")
})
3️⃣

Append File

Existing file-க்கு new content add பண்ண.

appendFile.js 
fs.appendFile("data.txt", "New content", (err) => {
  if (err) throw err
})
4️⃣

Delete File

unlink.js 
fs.unlink("data.txt", (err) => {
  if (err) throw err
  console.log("File deleted")
})
5️⃣

Create Folder

mkdir.js 
fs.mkdir("testFolder", (err) => {
  if (err) throw err
})

Sync vs Async Methods

Async Recommended

async 
fs.readFile()

Non-blocking.

Sync Method

sync 
fs.readFileSync()

Blocking operation.

Node.js normally async methods use பண்ணுவது better.

🔧 Real Project Use

Examples:

  • log files store பண்ண
  • configuration files read பண்ண
  • uploaded files manage பண்ண

HTTP Module

Tamil Explanation

HTTP module என்பது Node.js built-in module.


இதன் மூலம் web server create பண்ணலாம் மற்றும் client requests handle பண்ணலாம்.

HTTP server:

  • request receive பண்ணும்
  • response send பண்ணும்

Import HTTP Module

import 
const http = require("http")

Create HTTP Server

server.js 
const http = require("http")

const server = http.createServer((req, res) => {
  res.write("Hello from Node.js")
  res.end()
})

server.listen(5000, () => {
  console.log("Server running on port 5000")
})

Explanation

req
client request
res
server response
createServer
server create
listen
port run

Routing Example

routing.js 
const http = require("http")

const server = http.createServer((req, res) => {

  if (req.url === "/") {
    res.write("Home Page")
  }

  else if (req.url === "/about") {
    res.write("About Page")
  }

  res.end()
})

server.listen(5000)
