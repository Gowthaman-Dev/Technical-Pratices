Streams vs Buffers

Node.js Class Teaching Method - Complete Master Guide

1️⃣ Start with Real Life Example (Best Hook)

Teacher class-ல first question கேளுங்க.

1GB video file download பண்ணும்போது computer முழு file-யும் memory-ல load பண்ணுமா?

Students usually say Yes / No.

Then explain 👇

📌 Case 1 – Buffer

👉 Full data memory-ल store ஆகுது
👉 பிறகு process ஆகுது
Internet → Buffer (store full data) → Process

📌 Case 2 – Stream

👉 Data small chunks-ஆ வரும்
👉 வந்தவுடன் process ஆகும்
Internet → Chunk → Process
→ Chunk → Process
→ Chunk → Process
📌 இதுதான் Stream concept

2️⃣ Definition

Buffer

📌 Buffer என்பது binary data store பண்ண temporary memory area

  • files
  • images
  • videos
  • network packets

Stream

📌 Stream என்பது data-வை small chunks-ஆ process பண்ணும் mechanism

  • Full data memory-ல load ஆகாது
  • Chunk by chunk processing
  • More efficient
  • Lower memory usage

3️⃣ Simple Example

Buffer Example

Buffer Approach const fs = require("fs") fs.readFile("video.mp4",(err,data)=>{ console.log(data) })
👉 full file memory-ल load ஆகும்

Stream Example

Stream Approach const fs = require("fs") const stream = fs.createReadStream("video.mp4") stream.on("data",(chunk)=>{ console.log(chunk) })
👉 file chunk by chunk read ஆகும்

4️⃣ Comparison Table (Class Board)

Feature Streams Buffers
Data handling Chunk by chunk Full memory storage
Purpose Process data Store binary data
Memory usage Low Higher
Performance Fast for large files Not efficient for large data

5️⃣ Real World Usage

Streams use cases

🎬 video streaming
📦 large file download
🌐 network data transfer

Buffers use cases

📄 file reading
🖼️ image processing
📦 binary data storage

6️⃣ Important Concept (Very Interview Important)

Teacher class-ल சொல்லணும்:

👉 Streams internally buffers use பண்ணும்

Explanation:

Data chunk வரும்போது அது buffer-ல temporary store ஆகும்

💡 Key Point: Streams are built on top of buffers for efficient data handling!

7️⃣ Final Interview Answer (Teach Students)

Students memorize பண்ண easy-ஆ சொல்லுங்க:

In Node.js, streams and buffers are used for efficient data handling. Streams process data in small chunks without loading the entire data into memory, while buffers store binary data temporarily during data transfer.
Buffers store binary data temporarily, while streams process data chunk by chunk without loading the entire data into memory.
