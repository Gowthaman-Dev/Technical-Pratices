2️⃣ Buffer Creation

Node.js-ல் buffer create பண்ண multiple methods இருக்கு.

1️⃣ Buffer.from()

Existing data இருந்து buffer create பண்ணும்

Code

const buffer = Buffer.from("Hello") console.log(buffer)

Output

<Buffer 48 65 6c 6c 6f>

Tamil Explanation

"Hello" string
👉 binary format-ஆ convert ஆகுது
👉 buffer-ல store ஆகுது
48 65 6c 6c 6f இது hexadecimal representation

2️⃣ Buffer.alloc()

Memory allocate பண்ணும்

Code

const buffer = Buffer.alloc(10) console.log(buffer)

Explanation

👉 10 bytes memory create ஆகும்
👉 initially empty buffer

Output example

<Buffer 00 00 00 00 00 00 00 00 00 00>

3️⃣ Buffer.toString()

Buffer data string-ஆ convert பண்ணும்

Code

const buffer = Buffer.from("Node") console.log(buffer.toString())

Output

Node

Explanation

Buffer-ல இருக்கும் binary data → string-ஆ convert ஆகுது

⭐ Buffer Example

Code

const buffer = Buffer.from("Hello") console.log(buffer.length) console.log(buffer.toString())

Output

5 Hello

Explanation

buffer.length
👉 buffer size (bytes)
Hello = 5 characters → 5 bytes
