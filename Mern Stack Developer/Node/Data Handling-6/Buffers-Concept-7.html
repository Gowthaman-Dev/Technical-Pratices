2️⃣ Definition

📌 Tamil Explanation

Multiple buffers-ஐ ஒரே buffer-ஆ combine பண்ண Buffer.concat() method use பண்ணுறோம்.

📌 Simple English Definition

Buffer.concat() is used to combine multiple buffers into a single buffer.

3️⃣ Example Code

const buffer1 = Buffer.from("Hello ") const buffer2 = Buffer.from("Node") const result = Buffer.concat([buffer1, buffer2]) console.log(result.toString())

Output

Output: Hello Node
