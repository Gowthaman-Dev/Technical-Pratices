1️⃣ Streams
📌 Tamil Explanation
Stream என்பது 👉 data-ஐ small chunks (பகுதிகள்) ஆக process பண்ணும் method.
Large files handle பண்ணும்போது முழு file memory-க்கு load பண்ணாமல் piece by piece process பண்ணும்.
அதனால தான் Node.js high performance.
Example:
🎬Video streaming
📄File reading
🌐Network data transfer

⭐ Types of Streams (Interview Important)
1️⃣ Readable Stream

Data read பண்ண use பண்ணும்.

reading file HTTP request
2️⃣ Writable Stream

Data write பண்ண use பண்ணும்.

writing file HTTP response
3️⃣ Duplex Stream

👉 Read + Write இரண்டும் செய்யும்.

TCP socket
4️⃣ Transform Stream

👉 data read பண்ணி transform செய்து write பண்ணும்.

file compression encryption
1️⃣ Readable Stream — Example:
const fs = require("fs")

const readStream = fs.createReadStream("data.txt","utf8")

readStream.on("data",(chunk)=>{
 console.log(chunk)
})
2️⃣ Writable Stream — Example:
const fs = require("fs")

const writeStream = fs.createWriteStream("output.txt")

writeStream.write("Hello Node.js")
⭐ Stream Methods
MethodPurpose
pipe() connect streams
on("data") receive data
on("end") stream finished
Example:
const fs = require("fs")

const readStream  = fs.createReadStream("data.txt")
const writeStream = fs.createWriteStream("copy.txt")

readStream.pipe(writeStream)

2️⃣ Backpressure (Very Important Interview Question)
Tamil Explanation
Backpressure என்பது 👉 data producer fast-ஆ send பண்ணும் போது consumer slow-ஆ process பண்ணும் situation.
Streams automatically flow control handle பண்ணும்.
⚡ Backpressure Flow
Fast Data Source
↓
Slow Destination
Solution:
👉 stream buffering 👉 pipe() flow control
English sentence

"Backpressure occurs when the data producer is faster than the consumer, and streams handle it using flow control."

3️⃣ Flowing Mode vs Paused Mode
Readable streams 2 modes இருக்கும்.

data automatically flow

Paused

manual reading

Example:

readStream.resume()
4️⃣ HighWaterMark (Advanced but interview useful)
Tamil Explanation
Stream buffer size control பண்ணும் option.
Example:
const stream = fs.createReadStream("data.txt",{
  highWaterMark: 16 * 1024
})
👉 16KB chunk size.

2️⃣ Buffers
📌 Tamil Explanation
Buffer என்பது 👉 binary data store பண்ணும் temporary memory area.
Node.js-ல் binary data handle பண்ண buffers use பண்ணப்படும்.
Example:
📁file reading
🖼️image processing
📦network packets
⭐ Why Buffers Used
Binary data handle பண்ண:
🖼️images
🎬videos
📄files
🌐network data
⭐ Buffer Creation
1️⃣ Buffer.from() 

const buffer = Buffer.from("Hello")

console.log(buffer)
Output:
<Buffer 48 65 6c 6c 6f>
2️⃣ Buffer.alloc()

Memory allocate பண்ணும்.

const buffer = Buffer.alloc(10)

console.log(buffer)
3️⃣ Buffer.toString()

Buffer data convert to string.

const buffer = Buffer.from("Node")

console.log(buffer.toString())
Output:
Node
⭐ Buffer Example 
const buffer = Buffer.from("Hello")

console.log(buffer.length)
console.log(buffer.toString())
Output:
5
Hello

✅Network communication

⭐ Interview Sentence

3️⃣ Buffer Slice
Tamil Explanation
Buffer-ல இருக்கும் data-வின் ஒரு பகுதியை extract பண்ண use பண்ணப்படும்.

console.log(slice.toString())
Output:

Interview Sentence
