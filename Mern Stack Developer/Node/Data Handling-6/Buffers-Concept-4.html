Buffer Length

Node.js Class Teaching Method - Complete Master Guide

1️⃣ Start with Simple Question

Teacher class-ल first question கேளுங்க.

Hello என்ற word-ல எத்தனை characters இருக்கிறது?
Students answer: 5

அதுக்கப்புறம் explain பண்ணுங்க:

Node.js-ல் Buffer-ல data store ஆகும் 👉 அந்த stored data size-ஐ தெரிஞ்சிக்க length property use பண்ணுறோம்.

2️⃣ Definition

📌 Tamil Explanation

Buffer-ல store பண்ணிய data size (bytes) check பண்ண `buffer.length` use பண்ணுறோம்.

📌 Simple English Definition

Buffer length represents the size of the stored binary data in bytes.

3️⃣ Example Code

const buffer = Buffer.from("Hello") console.log(buffer.length)

Output

Output: 5

4️⃣ Step-by-Step Explanation (Class-ल சொல्ल)

1️⃣ "Hello" buffer-ஆ convert ஆகுது

Hello → Buffer

2️⃣ அந்த buffer-ல 5 characters store ஆகும்

5 characters are stored in the buffer

3️⃣ அதனால

buffer.length = 5

📌 Node.js-ल length → bytes count

5️⃣ Another Example (Students புரிய)

const buffer = Buffer.from("NodeJS") console.log(buffer.length)

Output

Output: 6

Explanation

NodeJS = 6 characters → 6 bytes
