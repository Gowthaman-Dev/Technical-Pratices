📦 NPM Deep Dive

NPM – Node Package Manager

What is NPM, install types, node_modules, commands deep dive — Tamil classroom guide.

1️⃣

First Idea

நீ ஒரு server create பண்ணணும் என்று imagine பண்ணு.

நீயே server code முழுவதும் எழுதலாம். ஆனா அது romba time ஆகும்.

அதனால் developers already ready-made code libraries create பண்ணியிருக்காங்க.

Example

  • Express → server create
  • Axios → API call
  • Mongoose → MongoDB connect

இந்த libraries-ஐ install பண்ண உதவுவது NPM.

2️⃣

NPM என்றால் என்ன?

NPM = Node Package Manager
Simple meaning:

👉 Node.js packages install மற்றும் manage பண்ணும் tool

3️⃣

Simple Real Life Example

NPM = Play Store / App Store மாதிரி.

📱 Phone-ல

  • WhatsApp install
  • Instagram install

💻 Node.js project-ல

  • express install
  • axios install
4️⃣

Example Command

terminal 
npm install express

இதன் meaning

👉 Express library download பண்ணு.

5️⃣

Install பண்ணும்போது என்ன நடக்கும்?

இந்த command run பண்ணும்போது 3 things நடக்கும்.

1️⃣

Express install ஆகும்

Server create பண்ண express code download ஆகும்.

2️⃣

node_modules folder create ஆகும்

Project folder-ல ஒரு new folder வரும்.

folder 
node_modules

இதில் இருக்கும்:

  • express package
  • express dependencies
3️⃣

package.json update ஆகும்

Project information save ஆகும்.

Example

package.json 
"dependencies": {
  "express": "^4.18.2"
}

👉 project express use பண்ணுது.

NPM Commands – Deep Dive
1️⃣

Install Package

Command

terminal 
npm install express

👉 Express library download பண்ணு

Express என்ன?

👉 Server create பண்ண உதவும் library.

Install ஆன பிறகு project folder-ல:

node_modules folder create ஆகும்.

2️⃣

Dev Dependency

Command

terminal 
npm install nodemon --save-dev

👉 Development time மட்டும் use பண்ணும் package

Example

  • nodemon → server auto restart
  • eslint → code errors check
  • prettier → code formatting

Production-ல தேவையில்லை.

Example

terminal 
nodemon app.js

Code change பண்ணினால் server automatically restart ஆகும்.

3️⃣

Global Package

Command

terminal 
npm install -g nodemon

👉 Package computer முழுவதும் install ஆகும்

அதனால் எந்த project-லவும் use பண்ணலாம்.

Example

terminal 
nodemon app.js
4️⃣

Remove Package

Command

terminal 
npm uninstall express

👉 express package remove ஆகும்.

5️⃣

Install All Dependencies

Imagine:

நீ GitHub-ல இருந்து project download பண்ணுற.

Project-ல:

project 
package.json

இருக்கும்.

இந்த file-ல dependencies list இருக்கும்.

Command run பண்ணு:

terminal 
npm install

👉 package.json-ல இருக்கும் all packages install ஆகும்

6️⃣

node_modules Folder

Package install பண்ணும்போது automatically create ஆகும்.

Example structure

project structure
project
│
├── node_modules
│     ├── express
│     ├── body-parser
│     └── other packages
│
├── package.json
└── app.js

இதில் இருக்கும்:

  • 1️⃣ Installed packages
  • 2️⃣ Packages dependencies

Example

Express install பண்ணினால் Express use பண்ணும் other libraries கூட install ஆகும்.
