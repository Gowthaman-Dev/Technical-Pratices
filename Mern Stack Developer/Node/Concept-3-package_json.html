Step 02

Definition

Board-ல எழுதணும்.

👉 package.json = Node.js project information file

Meaning — இந்த file-ல இருக்கும்
  • project name
  • version
  • dependencies
  • scripts
Step 03

Example File

Students-க்கு simple example காட்டணும்.

package.json 
{
  "name": "mern-project",
  "version": "1.0.0",
  "main": "server.js",
  "scripts": {
    "start": "node server.js"
  },
  "dependencies": {
    "express": "^4.18.2"
  }
}

Explain one-by-one.

Step 04

Important Fields Explain

name

Project name.

package.json 
"name": "mern-project"
version

Project version.

package.json 
"version": "1.0.0"
main

Project entry file.

package.json 
"main": "server.js"

👉 server start ஆகும் file.

scripts

Commands run பண்ண.

Example
package.json 
"scripts": {
  "start": "node server.js",
  "dev": "nodemon server.js"
}
Run command
terminal 
npm run dev

👉 nodemon server start.

dependencies

Project run ஆக தேவையான packages.

Example
package.json 
"dependencies": {
  "express": "^4.18.2"
}
Examples
  • express
  • mongoose
  • bcrypt
  • jsonwebtoken
devDependencies

Development time மட்டும் use பண்ணும்.

Examples
  • nodemon
  • eslint
