1️⃣1️⃣ Callback Functions
Tamil Explanation
Callback function என்றால் 👉 ஒரு function முடிந்த பிறகு execute ஆகும் function.
Node.js-ல asynchronous operations handle பண்ண callbacks அதிகமாக use பண்ணப்பட்டது.
Examples:
📄 File read 🌐 API call 🗄️ Database query
callback.js
function greet(name,callback){
  console.log("Hello " + name)
  callback()
}

function done(){
  console.log("Task completed")
}

greet("Gowtham",done)
Output
Hello Gowtham
Task completed

Real Node.js Example

readFile.js
const fs = require("fs")

fs.readFile("data.txt","utf8",(err,data)=>{
  if(err){
    console.log(err)
  }
  else{
    console.log(data)
  }
})
→ fs.readFile asynchronous
→ file read முடிந்த பிறகு callback run ஆகும்
⚠️ Callback Hell Problem

Multiple callbacks nested ஆகும் போது code messy ஆகும்.

callbackHell.js
login(user,function(){
  getData(function(){
    updateData(function(){
      saveData(function(){
      })
    })
  })
})

இதுதான் Callback Hell. Problems:

code readability குறையும்
debugging difficult
maintain difficult
→ இதற்கு solution → Promises
⭐ English Sentence (Interview)

"A callback function is a function passed as an argument to another function and executed after the operation completes."

1️⃣2️⃣ Promises
Tamil Explanation
Promise என்பது asynchronous operation-ஐ handle பண்ண modern method.
Promise future-ல் success அல்லது failure result return பண்ணும் object.

Promise States — Promise-க்கு 3 states இருக்கும்.

StateMeaning
Pendingoperation running
Fulfilledsuccess
Rejectederror

Promise Syntax

promise-syntax.js
const promise = new Promise((resolve,reject)=>{
  if(true){
    resolve("Success")
  }
  else{
    reject("Error")
  }
})

Using Promise

using-promise.js
promise
.then(result=>{
  console.log(result)
})
.catch(error=>{
  console.log(error)
})
fetchData.js
const fetchData = new Promise((resolve,reject)=>{
  setTimeout(()=>{
    resolve("Data received")
  },2000)
})

fetchData
.then(data=>console.log(data))
.catch(err=>console.log(err))

Advantages of Promises

✅Callback hell avoid
📖Code readable
🛡️Better error handling
⭐ English Sentence (Interview)

"A Promise is an object that represents the eventual completion or failure of an asynchronous operation."

1️⃣3️⃣ Async / Await
Tamil Explanation
Async/Await என்பது Promises-ஐ simple syntax-ல handle பண்ண modern method.
இது code-ஐ synchronous style போல readable-ஆ எழுத உதவும்.

Async Function

async keyword use பண்ணும் function always promise return பண்ணும்.
async-fn.js
async function getData(){
  return "Hello"
}

Await Keyword

await promise resolve ஆகும் வரை wait பண்ணும்.
await.js
async function fetchData(){

  const data = await Promise.resolve("Data received")

  console.log(data)
}

fetchData()

Example with API

api-call.js
async function getUsers(){

  try{
    const res = await fetch("https://api.example.com/users")
    const data = await res.json()

    console.log(data)
  }

  catch(err){
    console.log(err)
  }

}

Advantages

✨clean syntax
🐛easy debugging
📖readable code
⭐ English Sentence (Interview)

"Async/Await is a modern syntax built on top of promises that allows writing asynchronous code in a synchronous style."

1️⃣4️⃣ Event Loop
Tamil Explanation
Event Loop என்பது Node.js-ல் asynchronous operations handle பண்ணும் core mechanism.
⚡Single thread
🔄Event driven
🚀multiple requests handle
ஆனா multiple requests handle பண்ண event loop use பண்ணும்.

How Event Loop Works — Flow:

1Call Stack
↓
2Node APIs
↓
3Callback Queue
↓
4Event Loop
1️⃣
Call Stack Current function execution.
2️⃣
Node APIs Async operations run ஆகும்.
Examples: setTimeout · file system · network request
3️⃣
Callback Queue Completed callbacks wait ஆகும்.
4️⃣
Event Loop Call stack empty ஆனதும் queue-ல இருக்கும் callback execute பண்ணும்.
eventLoop.js
console.log("Start")

setTimeout(()=>{
  console.log("Async task")
},2000)

console.log("End")
Output
Start
End
Async task
→ setTimeout asynchronous
→ event loop later execute பண்ணும்

Why Event Loop Important

Event loop காரணம்:

✅ Scalable Node.js scalable
✅ High Volume thousands of requests handle
✅ Non-Blocking non blocking operations
⭐ English Sentence (Interview)

"The event loop is the mechanism that allows Node.js to perform non-blocking asynchronous operations by handling callbacks in the event queue."

5️⃣ setTimeout / setInterval
Tamil Explanation
setTimeout 👉 ஒரு function-ஐ delay செய்து execute பண்ணும்.
setInterval 👉 repeated interval-ல execute பண்ணும்.

Example

"setTimeout and setInterval are timer functions used to schedule asynchronous execution in Node.js."

6️⃣ setImmediate()
Tamil Explanation
setImmediate என்பது 👉 event loop-ல் current I/O phase முடிந்த உடனே execute ஆகும் function.

Example

setImmediate(()=>{
 console.log("Immediate task")
})
⭐ Interview Sentence

"setImmediate executes a callback immediately after the current I/O operations are completed."

7️⃣ process.nextTick()
Tamil Explanation
process.nextTick என்பது 👉 event loop-க்கு முன்னாடியே execute ஆகும் callback.
அதாவது queue-க்கு போகாமல் next tick-ல run ஆகும்.

Example

process.nextTick(()=>{
 console.log("Next tick")
})
⭐ Interview Sentence

"process.nextTick schedules a callback to be executed immediately after the current operation before the event loop continues."
