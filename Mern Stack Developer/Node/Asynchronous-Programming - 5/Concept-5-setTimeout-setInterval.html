Class Teaching Notes

setTimeout / setInterval

Step-by-step teaching · Tamil + English

1️⃣ Step 1 Start With Real Question
Class-ல first question கேள்:
🗣️ "ஒரு function-ஐ 5 seconds கழித்து run பண்ணணும்னா எப்படி?"
Students usually answer தெரியாது.
👉 JavaScript / Node.js-ல Timer Functions use பண்ணுவோம்.
Two main timers:
setTimeout setInterval
2️⃣ Step 2 Introduce setTimeout
Board-ல definition:
Definition
👉 setTimeout = delay செய்து function run பண்ணும்
Syntax:
setTimeout(function, delay)
Example:
setTimeout(()=>{
 console.log("Hello")
},2000)
Explain:
PartMeaning
functionrun ஆகும் code
2000 milliseconds delay
Important:
1000 ms = 1 second
So:
2000 ms = 2 seconds
3️⃣ Step 3 Execution Flow
Code:
console.log("Start")

setTimeout(()=>{
 console.log("Hello")
},2000)

console.log("End")
Output:
Start
End
Hello
Explain:
👉 setTimeout asynchronous
👉 event loop later execute பண்ணும்.
4️⃣ Step 4 Introduce setInterval
Board-ல definition:
Definition
👉 setInterval = repeated interval-ல function run பண்ணும்
Syntax:
setInterval(function, delay)
Example:
setInterval(()=>{
 console.log("Running...")
},2000)
Output:
Running...
Running...
Running...
Every 2 seconds repeat ஆகும்.
5️⃣ Step 5 Stop Interval (Important)
Students-க்கு இது miss ஆகக்கூடாது.
Example:
const timer = setInterval(()=>{
 console.log("Hello")
},1000)

setTimeout(()=>{
 clearInterval(timer)
},5000)
Explain:
1 second Hello
2 second Hello
3 second Hello
4 second Hello
5 second stop
clearInterval() → interval stop பண்ணும்.
6️⃣ Step 6 Difference
Board-ல table.
setTimeoutsetInterval
run once repeat
delay execution repeated execution
Example:
setTimeout
setTimeout → run one time
setInterval
setInterval → run many times
7️⃣ Step 7 Real Life Example
Alarm example:
⏰ Alarm Examples
⏱️ setTimeout → alarm after 5 seconds
🕐 setInterval → clock tick every second
8️⃣ Event Loop Connection
Explain:
🔄 Event Loop Flow
👉 setTimeout  and  setInterval  Node APIs → Callback Queue → Event Loop.
setTimeout / setInterval
↓
Node APIs
↓
Callback Queue
↓
Event Loop
So they are asynchronous timers.
9️⃣ Interview Sentence
Students memorize:
⭐ Interview Sentence

"setTimeout and setInterval are timer functions used to schedule asynchronous execution in Node.js."

👉 "What is the difference between setTimeout(fn,0) and normal function execution?"
