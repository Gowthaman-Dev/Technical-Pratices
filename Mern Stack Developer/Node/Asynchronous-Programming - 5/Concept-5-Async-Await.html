Async / Await – Class Guide

Step-by-step teaching · Tamil + English

1️⃣ Step 1 Start With Problem (Promises Code)
First students-க்கு Promise code remind பண்ணு.
login()
.then(getData)
.then(updateData)
.then(saveData)
.catch(err=>console.log(err))
Explain:
👉 இது Promise chaining
👉 Code readable ஆனாலும் இன்னும் .then() .then() இருக்கும்.
👉 இதை இன்னும் simple ஆ எழுத Async / Await வந்தது.

4️⃣ Step 4 Await Keyword
Students-க்கு most important concept இது.
Example:
async function fetchData(){

  const data = await Promise.resolve("Data received")

  console.log(data)
}

fetchData()
Explain step by step:
fetchData start
↓
await promise
↓
promise resolve
↓
data received
↓
console.log
Meaning:
👉 await promise complete ஆகும் வரை wait பண்ணும்.
5️⃣ Step 5 Real Example (API Call) 
async function getUsers(){

  try{
    const res  = await fetch("https://api.example.com/users")
    const data = await res.json()

    console.log(data)
  }

  catch(err){
    console.log(err)
  }

}
Explain flow:
API request
↓
response wait
↓
data convert JSON
↓
print data
Important:
CodeMeaning
await fetch() API response wait
await res.json()JSON convert

7️⃣ Step 7 Promise vs Async/Await
Board-ல comparison.
PromiseAsync/Await
.then() await
chaining sequential code
less readable very readable
Example:
Promise:
login()
.then(getData)
.then(updateData)
.then(saveData)
Async/Await:
async function process(){
 await login()
 await getData()
 await updateData()
 await saveData()
}
👉 Code normal synchronous code போல இருக்கும்.
