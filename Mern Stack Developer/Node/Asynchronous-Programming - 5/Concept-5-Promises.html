5️⃣ Step 5 Real Async Example 
const fetchData = new Promise((resolve,reject)=>{
  setTimeout(()=>{
    resolve("Data received")
  },2000)
})

fetchData
.then(data=>console.log(data))
.catch(err=>console.log(err))
Explain flow:
start
↓
2 seconds wait
↓
resolve()
↓
then() run
↓
Data received
6️⃣ Step 6 Promise vs Callback
Board-ல comparison எழுதணும்.
CallbackPromise
nested functionschaining
messy code clean code
hard debugging easy debugging
Example:
Callback:
login(()=>{
 getData(()=>{
  updateData(()=>{
  })
 })
})
Promise:
login()
.then(getData)
.then(updateData)
.then(saveData)
👉 Code clean ஆகும்.
