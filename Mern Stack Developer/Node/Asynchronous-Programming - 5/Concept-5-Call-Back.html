Class Teaching Notes

Callback Functions

Step-by-step class guide · Tamil + English

1️⃣ Step 1 Simple Question
Class start பண்ணும்போது கேள்:
🗣️ "ஒரு task முடிந்த பிறகு next task run ஆகணும்னா எப்படி?"
Example:
file read முடிந்த பிறகு log print database query முடிந்த பிறகு response send
👉 இதற்கு Callback Function use பண்ணுவோம்.
2️⃣ Step 2 Definition
Board-ல எழுதணும்:
Definition
👉 Callback = Function inside another function
Simple meaning:
👉 ஒரு function-க்கு argument ஆக pass பண்ணப்படும் function.
3️⃣ Step 3 Simple Example
Students-க்கு easy example காட்டணும்.
function greet(name,callback){
 console.log("Hello " + name)
 callback()
}

function done(){
 console.log("Task completed")
}

greet("Gowtham",done)
Explain:
1️⃣ greet() function run ஆகும்
2️⃣ "Hello Gowtham" print ஆகும்
3️⃣ callback() run ஆகும்
4️⃣ done() execute ஆகும்
Output:
Hello Gowtham
Task completed
4️⃣ Step 4 Real Node.js Example
Node.js-ல callbacks அதிகமாக use ஆகும்.
Example:
const fs = require("fs")

fs.readFile("data.txt","utf8",(err,data)=>{
 if(err){
  console.log(err)
 }
 else{
  console.log(data)
 }
})
Explain:
👉 fs.readFile() asynchronous operation.
Meaning:
1️⃣ file read start ஆகும்
2️⃣ file read complete ஆன பிறகு
3️⃣ callback function run ஆகும்.
5️⃣ Step 5 Callback Hell
Class-ல biggest problem explain பண்ணணும்.
Example:
login(user,function(){
 getData(function(){
  updateData(function(){
   saveData(function(){
   })
  })
 })
})
Explain:
👉 function inside function inside function.
Diagram:
login
↓
getData
↓
updateData
↓
saveData
இதுதான் Callback Hell.
6️⃣ Problems of Callback Hell
Board-ல எழுதணும்:
code readability குறையும்
debugging difficult
maintain difficult
7️⃣ Solution
Modern Solution
👉 Promises 👉 Async/Await
Modern Node.js mostly Promises / Async Await use பண்ணும்.
8️⃣ Real Life Example
🍽️ Restaurant Example
Customer order → cook food → waiter serve.
Sequence:
order
↓
cook
↓
serve
👉 ஒவ்வொரு step முடிந்த பிறகு next step run ஆகும். இதுதான் callback idea.
9️⃣ Interview Sentence
Students memorize this:
⭐ Interview Sentence

"A callback function is a function passed as an argument to another function and executed after the operation completes."

1️⃣ First Concept – Callback என்ன?
Students-க்கு first இந்த simple code காட்டணும்.
function greet(name, callback){
  console.log("Hello " + name)
  callback()
}

function done(){
  console.log("Task completed")
}

greet("Rocky", done)
Flow explain:
greet() run
↓
Hello Rocky print
↓
callback() run
↓
done() execute
Output:
Hello Rocky
Task completed
👉 Meaning: ஒரு function முடிந்த பிறகு next function run ஆகுது.
இதுதான் callback.
2️⃣ Real Node.js Example (fs.readFile)
Code:
const fs = require("fs")

fs.readFile("data.txt","utf8",(err,data)=>{
  if(err){
    console.log(err)
  }
  else{
    console.log(data)
  }
})
Students-க்கு step by step explain:
Step 1
fs.readFile()
Node.js file read start பண்ணும்.
Step 2
File read ஆக time ஆகும்.
Node.js wait பண்ணாது.
Step 3
File read complete ஆன பிறகு
(err,data)=>{}
இந்த callback function run ஆகும்.
Diagram:
readFile start
↓
file reading
↓
file read complete
↓
callback function run
👉 அதனால் தான் callback use பண்ணுறோம்.
3️⃣ Callback Hell என்ன?
Students-க்கு இந்த question கேள்:
🗣️ "ஒரு task முடிந்த பிறகு next task run ஆகணும்னா?"
Example:
login
↓
getData
↓
updateData
↓
saveData
Code:
login(user,function(){
  getData(function(){
    updateData(function(){
      saveData(function(){
      })
    })
  })
})
Diagram:
login(
   getData(
      updateData(
         saveData()
      )
   )
)
👉 function inside function inside function.
இதுதான் Callback Hell.
4️⃣ Why Callback Hell Problem?
Problems:
1️⃣ Code deep nesting
{
 {
  {
   {
   }
  }
 }
}
2️⃣Read பண்ண difficult
3️⃣Debug difficult
4️⃣Maintain difficult
5️⃣ Real Life Example
🍽️ Restaurant Example
order
↓
cook
↓
serve
↓
payment
👉 ஒவ்வொரு step முடிந்த பிறகு next step run ஆகும்.
👉 Code-ல இதை callbacks use பண்ணினா nested ஆகும்.
6️⃣ Solution
Callback Hell avoid பண்ண modern JavaScript use பண்ணும்:
Modern Alternatives
👉 Promises 👉 Async / Await
Example:
login()
.then(getData)
.then(updateData)
.then(saveData)
Code clean ஆகும்.
