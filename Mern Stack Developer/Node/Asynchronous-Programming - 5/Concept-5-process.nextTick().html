Class Teaching Notes

process.nextTick() – Class Guide

Step-by-step teaching · Tamil + English

1️⃣ Step 1 Start With Simple Question
Class-ல கேள்:
🗣️ "Current code finish ஆன உடனே next task run பண்ணணும்னா எப்படி?"
Options explain பண்ணு:
setTimeout setImmediate process.nextTick
👉 இதில் most fastest execution → process.nextTick()
2️⃣ Step 2 Simple Definition
Board-ல எழுதணும்:
Definition
👉 process.nextTick() = current operation முடிந்த உடனே run ஆகும் callback
Important idea:
Current code finish
↓
nextTick callback run
↓
event loop continue
Meaning:
👉 Event loop wait பண்ணாது.
3️⃣ Step 3 Basic Syntax 
process.nextTick(callback)
Example:
process.nextTick(()=>{
 console.log("Next tick")
})
Explain:
👉 Node.js next tick phase-ல execute பண்ணும்.
4️⃣ Step 4 Simple Example 
console.log("Start")

process.nextTick(()=>{
 console.log("Next tick")
})

console.log("End")
Output:
Start
End
Next tick
Explain flow:
Start
↓
nextTick register
↓
End
↓
nextTick execute
5️⃣ Step 5 Event Loop Relationship
Diagram students-க்கு explain பண்ணு:
Current code
↓
process.nextTick queue
↓
Event Loop
Meaning:
👉 nextTick event loop-க்கு முன்னாடி run ஆகும்.
6️⃣ Step 6 Compare With setImmediate
Example:
setImmediate(()=>{
 console.log("Immediate")
})

process.nextTick(()=>{
 console.log("NextTick")
})
Output usually:
NextTick
Immediate
Reason:
→ process.nextTick → before event loop
→ setImmediate → after event loop
7️⃣ Step 7 Execution Priority
Board-ல priority order எழுதணும்.
1️⃣ process.nextTick fastest
2️⃣ setImmediate
3️⃣ setTimeout
Meaning:
👉 nextTick fastest callback execution.

Restaurant example:
🍽️ Restaurant Example

9️⃣ When Used
Examples:
🛡️error handling
🧹cleanup tasks
⚙️scheduling small operations
🔟 Interview Sentence
Students memorize:
⭐ Interview Sentence

"process.nextTick schedules a callback to run immediately after the current operation, before the event loop continues."
