Modules

Tamil Explanation

Module என்றால் ஒரு JavaScript file அல்லது reusable code block.


Node.js-ல் code-ஐ small files ஆக split பண்ணி organize பண்ணலாம். இதனால்:

  • Code clean ஆக இருக்கும்
  • Reuse பண்ண முடியும்
  • Maintain easy ஆகும்

Node.js-ல் இரண்டு type modules இருக்கும்:

1️⃣ Core Modules

Node.js built-in modules.

Examples

  • fs → file system
  • http → server create
  • path → file paths
  • os → operating system info

Example

core.js 
const fs = require("fs")

2️⃣ User Defined Modules

நாமே create பண்ணும் modules.

Example

math.js
math.js 
function add(a, b) {
  return a + b
}

module.exports = add
app.js
app.js 
const add = require("./math")

console.log(add(2, 3))

Module Export Methods

1️⃣ module.exports

export 
module.exports = function() {}

2️⃣ exports

export 
exports.add = add

Why Modules Important

Code reuse Maintainability Separation of concerns Large applications manage easy
⭐ English Sentence (Interview)
"Modules in Node.js help organize code into reusable and maintainable files."

Global Objects

Tamil Explanation

Global Objects என்றால் Node.js application முழுவதும் access பண்ணக்கூடிய objects.


இதற்கு import தேவையில்லை.

Examples

console
process
setTimeout
setInterval
__dirname
__filename

Example

global.js 
console.log("Hello")

இது global object. Import தேவையில்லை.

Example 2

dirname.js 
console.log(__dirname)
▶ Output
project folder path

Important Global Objects

Object Purpose
consolelog messages
processprocess info
__dirnamecurrent folder path
__filenamecurrent file path
setTimeoutrun code later
⭐ English Sentence (Interview)
"Global objects are available throughout the Node.js application without requiring import."
6️⃣

Process Object

Tamil Explanation

process object என்பது Node.js application run ஆகும் current process information கொடுக்கும்.


இது ஒரு global object.

இதன் மூலம்:

  • environment variables access
  • command line arguments
  • process control

Example 1 (Environment Variables)

env.js 
console.log(process.env.PORT)

இது .env file values access பண்ண உதவும்.

Example 2 (Command Line Arguments)

argv.js 
console.log(process.argv)

Run command:

terminal 
node app.js hello
▶ Output
['node', 'app.js', 'hello']

Example 3 (Exit Process)

exit.js 
process.exit()

Program stop ஆகும்.

Important Process Properties

Property Purpose
process.envenvironment variables
process.argvcommand arguments
process.exit()stop program
process.pidprocess id
⭐ English Sentence (Interview)
"The process object provides information and control over the current Node.js process."
⭐

Final Interview Answer (Short)

நீங்க interview-ல சொல்லலாம்:

"Node.js uses modules to organize code into reusable files. Global objects are available throughout the application without importing them. The process object provides information about the currently running Node.js process, including environment variables and command-line arguments."
