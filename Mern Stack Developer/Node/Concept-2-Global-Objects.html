🌐 Global Objects

Node.js Global Objects – Teaching Guide

console, process, __dirname, setTimeout மற்றும் more — Tamil classroom guide with examples.

01

Step 1 – First Question (Start Class)

Class-ல first கேள்:

🗣️

"fs module use பண்ணணும்னா என்ன செய்யணும்?"

Students answer

👉 require("fs")

Example

import.js 
const fs = require("fs")

Explain:

👉 module use பண்ண import செய்யணும்

02

Step 2 – Next Question

அடுத்த question கேள்:

🗣️

"console.log use பண்ண import பண்ணுறோமா?"

Students

👉 No

Example

global.js 
console.log("Hello")

Explain:

👉 import இல்லாம use பண்ணுறோம்.

அதனால் இதை Global Object சொல்வாங்க.

03

Step 3 – Definition

Board-ல எழுதணும்:

👉 Global Objects = Import இல்லாம application முழுவதும் use பண்ணக்கூடிய objects

Tamil meaning

👉 Node.js program முழுவதும் direct-ஆ access பண்ணலாம்.

04

Important Global Objects

Students remember பண்ண இந்த list சொல்லு.

📋

1️⃣ console

Purpose

log messages print பண்ண.

Example

console.js 
console.log("Hello Node")
▶ Output
Hello Node
⚙️

2️⃣ process

Purpose

current Node.js process info.

Example

process.js 
console.log(process.platform)
▶ Output example
win32
📁

3️⃣ __dirname

Purpose

current folder path.

Example

dirname.js 
console.log(__dirname)
▶ Output
C:\project\app
📄

4️⃣ __filename

Purpose

current file path.

Example

filename.js 
console.log(__filename)
⏱️

5️⃣ setTimeout

Purpose

code later run பண்ண.

Example

timeout.js 
setTimeout(() => {
  console.log("Hello")
}, 2000)

👉 2 seconds பிறகு run ஆகும்.

🔁

6️⃣ setInterval

Purpose

repeated execution.

Example

interval.js 
setInterval(() => {
  console.log("Running")
}, 1000)

👉 every second run ஆகும்.

06

Real Teaching Trick

இந்த question கேள்:

🗣️

"Which is global?"

Example

compare.js 
fs.readFile()
console.log()

❌ requires import

import needed 
const fs = require("fs")
fs.readFile()

👉 fs module import செய்யணும்

✅ global — no import

no import 
console.log("Hello")

👉 console global

Students immediately புரியும்.

Interview Sentence

⭐ Interview Sentence
