fs

File System Module

Definition
fs — File System Module

Node.js built-in module — install செய்ய வேண்டாம். Server-ல் files மற்றும் folders-ஐ create, read, write, delete பண்ண use பண்ணுவோம்.

Import
const fs = require("fs")  // built-in — no install needed
5 Important Methods
01
fs.readFile()
Read File
File content read பண்ண
02
fs.writeFile()
Write File
New file create / content write பண்ண
03
fs.appendFile()
Append File
Existing file-க்கு content add பண்ண
04
fs.unlink()
Delete File
File delete பண்ண
05
fs.mkdir()
Create Folder
New folder / directory create பண்ண
Code Examples
readFile
fs.readFile("data.txt", "utf8",
  (err, data) => {
  if (err) throw err
  console.log(data)
  // file content print ஆகும்
})
writeFile
fs.writeFile("data.txt",
  "Hello Node.js",
  (err) => {
  if (err) throw err
  console.log("File created")
})
unlink (delete)
fs.unlink("data.txt",
  (err) => {
  if (err) throw err
  console.log(
    "File deleted")
})
Sync vs Async — Important Concept
Recommended
fs.readFile()
Async — Non-blocking. Server wait பண்ணாது. Other tasks continue ஆகும்.
Avoid
fs.readFileSync()
Sync — Blocking. File read ஆகும் வரை server wait பண்ணும். Performance drop.

Node.js-ல் always Async methods use பண்ணுவோம் — non-blocking performance-க்காக.

http

HTTP Module

Definition
http — built-in module

Node.js-ல் web server create பண்ண உதவும் built-in module. Install செய்ய வேண்டாம். Client requests receive பண்ணி responses send பண்ணும்.

const http = require("http")  // built-in — no install
Request / Response Flow
Browser
request →
Node.js HTTP Server
→ response
Browser

Browser → Request → Server → Response → Browser. இதுதான் HTTP communication.

Create HTTP Server
const http = require("http")

const server = http.createServer((req, res) => {
  res.write("Hello from Node.js")
  res.end()
})

server.listen(5000, () => {
  console.log("Server running on port 5000")
})
CodeMeaning
createServer()Server create பண்ணும். req + res callback கொடுக்கும்.
reqClient request info — URL, method, headers
resServer response — res.write(), res.end()
listen(5000)Port 5000-ல் server run ஆகும் — localhost:5000
Routing Example
const server = http.createServer(
  (req, res) => {
  if (req.url === "/") {
    res.write("Home Page")
  } else if (req.url ===
    "/about") {
    res.write("About Page")
  }
  res.end()
})
/ Home Page return ஆகும்
/about About Page return ஆகும்
HTTP Methods
MethodPurposeExample
GET Data get பண்ண GET /users
POST Data create பண்ண POST /users
PUT Data update பண்ண PUT /users/1
DELETE Data delete பண்ண DELETE /users/1
Why Express? — HTTP module limitation
HTTP module — complex
if (req.url === "/users"
  && req.method ===
  "GET") {
  // manual routing
}
Express — easy
app.get("/users",
  (req, res) => {
  // clean routing
})

HTTP module-ல் routing complex ஆகும் — அதனால் real projects-ல் Express framework use பண்ணுவோம்.
