Step 2 – Introduce HTTP Module

Board-ல எழுதணும்:

👉 Important: install செய்ய வேண்டாம் 👉 Node.js already provide பண்ணும்.
👉 Import:

3️⃣

Step 3 – Create Simple Server

Students-க்கு basic server code show பண்ணு.
const http = require("http")

const server = http.createServer((req,res) => {
res.write("Hello from Node.js")
res.end()
})

server.listen(5000,() => {
console.log("Server running on port 5000")
})
4️⃣

Step 4 – Code Explain

createServer() server create பண்ணும்.
http.createServer()

req client request information.
Example: URL, method, headers

res server response send பண்ண.
Example: res.write() , res.end()

listen() server எந்த port-ல run ஆகணும்.
Example: server.listen(5000)

👉 Meaning: localhost:5000
5️⃣

Step 5 – Routing Concept

Students-க்கு routing example காட்டணும்.
const http = require("http")

const server = http.createServer((req,res) => {
if(req.url === "/"){
res.write("Home Page")
}
else if(req.url === "/about"){
res.write("About Page")
}
res.end()
})

server.listen(5000)
Explain:
/ → Home Page
/about → About Page
