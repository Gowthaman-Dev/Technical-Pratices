03

Step 3 – Types of Modules

Node.js-ல 2 types modules இருக்கும்.

1️⃣ Core Modules

Node.js already provide பண்ணும் modules.

Examples

  • fs → file system
  • http → server create
  • path → file paths
  • os → operating system info

Example code

core.js 
const fs = require("fs")

Explain:

👉 require() use பண்ணி module import பண்ணுறோம்.

2️⃣ User Defined Modules

நாமே create பண்ணும் modules.

Example

math.js
math.js 
function add(a, b) {
  return a + b
}

module.exports = add

👉 function export பண்ணுறோம்.

app.js
app.js 
const add = require("./math")

console.log(add(2, 3))

👉 require() use பண்ணி module import பண்ணுறோம்.

▶ Output
5
04

Module Export Methods

Node.js-ல export பண்ண இரண்டு method இருக்கும்.

1️⃣ module.exports

export 
module.exports = add

👉 single function export பண்ணலாம்.

2️⃣ exports

export 
exports.add = add

👉 multiple functions export பண்ணலாம்.

Example

multi-export.js 
exports.add = add
exports.sub = sub
