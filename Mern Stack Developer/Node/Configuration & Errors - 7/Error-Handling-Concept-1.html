3️⃣

Types of Errors

3 Types

Node.js / JavaScript-ல் 3 main types of errors இருக்கிறது.

1️⃣ Syntax Error
📌 Meaning

Code syntax mistake.
Program run ஆகவே முடியாது.

Example
syntax-error.js
const name =
Explanation

Variable assign செய்யவில்லை.
Node.js run பண்ணும்போது syntax error throw ஆகும்.

Example error
Error
SyntaxError: Unexpected end of input
📌 Important Point Syntax error வந்தா:
👉 program start ஆகாது
2️⃣ Runtime Error
📌 Meaning

Program run ஆகும்போது error வரும்.

Example
runtime-error.js
const user = undefined

console.log(user.name)
Explanation

user = undefined
user.name access பண்ண முடியாது

Error
Error
TypeError: Cannot read property 'name'
📌 Important Point Runtime errors usually occur:
  • API calls
  • Database queries
  • Object access
3️⃣ Logical Error
📌 Meaning

Code run ஆகும் ஆனால் wrong output வரும்.

Example
logical-error.js
const result = 2 + 2

console.log(result === 5)

Expected: 4
But logic mistake இருந்தா output wrong.

Example explanation

2 + 2 = 4
But program logic wrong → result incorrect

📌 Note Logical errors debugging மூலம் கண்டுபிடிக்க வேண்டும்.

6️⃣

Basic Error Handling Example

Code

Node.js-ல் errors handle பண்ண try...catch use பண்ணலாம்.

try-catch.js
try {
  const user = undefined
  console.log(user.name)
} catch (error) {
  console.log("Error occurred:", error.message)
}
Output
Error occurred: Cannot read property 'name'
📌 Explanation
try
→ risky code
catch
→ error handle
